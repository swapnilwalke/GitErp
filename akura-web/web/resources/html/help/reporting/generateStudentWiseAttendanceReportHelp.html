Student Attendance Reports - Generate Student Wise Attendance Report

Welcome to the Akura Management System. Using this page you can generate student wise attendance report.

Student Wise Attendance
  • Enter the following details:
    • Select Class
    • Select Admission No
    • Select Date From
    • Select Date To
    • Fields marked with * are mandatory to fill.
  • Click 'Generate Report' button
