Student Attendance

Welcome to the Akura Management System. Using this page you can enter class wise attendance manually .

Student Attendance
  • Enter the following details:
    • Select Class
    • Select date
    • Fields marked with * are mandatory to fill.
  • Click 'Search Class' button to get a list of students in the particular class
  • Mark attendance using check boxes
  • Click 'Save' button to save attendance details
  • Click 'Check_All' button to check all the check boxes
  • Click 'Cancel' button to cancel the operation
