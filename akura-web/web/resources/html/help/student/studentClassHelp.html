Assign Students to Classes

Welcome to the Akura Management System. Using this page you can Assign student to a class

  • Search Students
  • Assign to class


Search Students
  • Please select the past grade using Select Grade drop down. You can either select New students or exiting grade from the drop down
  • Select the past year using the Select Year drop down.
  • Then Click the Search Student button to search for the student Given criteria.
  • Students will be listed at the Students in () list box


Assign to class
  • Select the students from Students in () list box
  • Then click on right arrow to move the students
  • The selected students will be moved to Selected Students
  • If you wants to delete students from the selected list you can use left arrow button
  • Then Select the class from Assign to Class drop down
  • Select the year from Assign to Year drop down
  • Clock on Save button to assign the selected students to Selected class and year
