Add Assignment Marks to Students

Welcome to the Akura Management System. Using this page you can enter assignment marks for students


  • Search Students
  • Enter Assignment Marks


Search Students
  • Please select the class using Select Class drop down.
    (All the assignments under the selected class will be shown in Select Assignment drop down)
  • Select the assignment using the Select Assignment drop down.
  • Students and the Grading/Marks will be Shown in the below table.


Enter Assignment Marks
  • If it's Marks - you have to enter marks as Percentage
  • If it's Grading - you have to enter Gradings such as A, B, etc.
  • Click on Save button to save the changes
