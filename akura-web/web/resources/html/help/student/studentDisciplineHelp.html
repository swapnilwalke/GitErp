Student Discipline

Welcome to the Akura Management System. Using this page you can Add , Delete and Edit student discipline details.

  • Add Student Discipline details
  • Edit student Discipline details
  • Delete Student Discipline Details


Student Discipline - Add Student Discipline details
  • Click 'Add Icon ' in the Student Discipline page and enter details for following:
    • Enter the details for all the fields marked as *
    • Enter Comment Value
    • Select Informed parents
    • Select Warning level
    • Select Date
    • Click save button to save discipline details or click cancel button to cancel details


    • Student Discipline - Edit student Discipline details
    • If you want to edit student discipline details , Edit function can be used.
    • Click on the 'Edit' Icon
      • Edit Comment details
      • Edit Informed parents
      • Edit Warning level
      • Edit Date
      • Click save button to save newly edited discipline details or click cancel button to cancel edited details


    Student Discipline - Delete Student Discipline Details
    • Select the record you want to delete and click delete icon on the record.
    • Click 'Ok' button to delete record from the system.
    • Click 'Cancel' to cancel the deletion.
