Add Marks to Students

Welcome to the Akura Management System. Using this page you can enter marks for the students



once you assign subjects to students, you can enter marks for Them

  • Search Class
  • Enter Term Marks
  • Enter Sub Term/Monthly Marks


Search Class
  • Please select the class using Select Class drop down.
  • Select the term using the Select Term drop down.
  • Then Select the Monthly Grades or Term Grades from the Select Grade drop down.
  • Students and the subjects allocated for the Students will be Shown in the below table.


Enter Term Marks
  • If you choose Term Grades - you have to enter marks as Percentage
  • Click on Save button to save the changes


Enter Sub Term/Monthly Marks
  • If you choose Monthly Grades - you have to enter marks as Grades (A,B,C,etc). You will be given the chance to enter marks for all the available sub teams you have added under reference module
  • Click on Save button to save the changes
