Student Details

Using this page you can do the registration of student to the system.



  • Complete the "General Information" section. All of the fields noted with an asterisk (*) are required.
  • Before adding a student make sure that the required reference data (Religion, Study Medium, Nationality, School House, Method of Travel and City information) have been entered to the system from the Reference Module.
  • Use calendar button to select a date.
  • Complete the Previous School Details, Disabilities and Emergency details if required. (Not Mandatory)
  • Click the <Reset> button to Reset the student details page.
  • Click the <Save> button to Save the student details.
  • Click the <Cancel> button to Navigate away from the student details page.
  • <Home> link to redirect to the "Home" page.
  • <Student Search> link to redirect to the "Student Search" page.
  • <Print Student Summary> link to view and print the Student Summary Report.
