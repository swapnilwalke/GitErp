Welcome to the Akura Management System. Using this page you can search students for the given criteria

  • Search
  • Advanced Search
  • Add New Student


Please click 'Search' for the regular search of students.



Search - Regular Search Procedure
  • You can search students by providing one or many criteria of following data.
    • Last Name
    • Admission No
    • Grade
    • Student Status
  • You can search students by using the last name of the student. It can be entire name of the last name or any part of the last name.
  • You can search students by using the admission number or grade of the student. If the new student ,you can search by using the "new student" under grade.
  • You can search students by using the status("Current","Past","Suspended" & "Temporary Leave") of the student.
    • Current - Students who are not assigned to a class/Grade and Present students, can be found under "Current" status.
    • Past - Students who left the School/Institute.
    • Suspended - Suspended students due to the disciplinary reasons.
    • Temporary Leave - Students who are on leave temporary.
  • Click the 'Search' button.
  • You can edit student details by navigating through the student search results.


Advanced Search - Advanced Search Procedure
  • If you want to search all the students that study the given name of a subject, Advanced Search can be used.
  • Click on the 'Advance Search' Button
  • Likewise by providing one or many fields of data of following search can be done.
    • Class
    • Sport
    • Subject
    • Prefect Type
    • Parent's Profession
  • Then click 'Search' button that is below of the 'Advanced Search' button.
