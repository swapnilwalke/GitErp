Academic Life

Using this page you can manage student Academic Life information for the selected student.



  • If there are any existing academic life details, the data will be display on the page.
  • All of the fields noted with an asterisk (*) are required.
  • Before adding the Prefect information and Scholarship information make sure that the required reference data has been entered to the system from the Reference Module.
  • Attendance progress bar is to display a summary of Attendance details of the student for the selected year.
  • Religious Activities progress bar is to display a summary of Religious Activities details of the student for the selected year.
  • Academic Life progress bar is to display a summary of academic life details of the student for the selected year.
  • <Home> link to redirect to the "Home" page.
  • <Student Search> link to redirect to the "Student Search" page.


    • Subject Marks

    In this section, you can view the followings

    • Student Term Marks with the student total marks and average marks.
    • Student Sub Term Marks by click on a subject link
    • Student Assignment Marks by click on a icon next to Subject
    • Student Exam Marks by click on a icon with View Exam Marks Details


    Prefect Details

      Add Prefect Details
    • To add prefect details use the Add button and fill the required details, then click Save.


      • Edit Prefect Details
    • To edit information in this page, click the Edit button in the row section. Make changes to the appropriate fields, then click Save .


      • Delete Prefect Details
    • To delete information in these page, click the Delete button in the row section. Confirm the message to delete the record either using Ok or Cancel.
