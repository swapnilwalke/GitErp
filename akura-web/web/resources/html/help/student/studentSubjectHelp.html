Assign Subjects to Students

Welcome to the Akura Management System. Using this page you can Assign subjects to students.



Once you assign students to classes, you can assign subjects to them.

  • Search Class
  • Assign subjects to students


Search Class
  • Please select the grade using Select Grade drop down.
  • Select the year using the Select Year drop down.
  • Then Click the Search Class button to search for the students Given criteria.
  • Students and the subjects allocated for the selected grade will be shown below table with the current subject allocations.


Assign subjects to students
  • You can assign the subjects to students using the check boxes
  • if you wants to remove subject from student you can remove tick from check box
  • If you wants to assign all the subjects for all the students you can use Check All button for that
  • Click on Save button to save the changes
