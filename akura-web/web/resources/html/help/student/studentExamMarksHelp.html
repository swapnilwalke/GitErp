Add Exam Marks to Students

Consider When adding exams it should be seperated for different scheams. Eg. For A/L , A/L Science, A/L Commerce, A/L Arts


Then from the 'Exam Subjects' UI allocate subjects for that exams seperately.


Welcome to the Akura Management System. Using this page you can enter marks for the students



Then the names of the students will be displayed for the particular class grade.

  • Search Students for the Grade
  • Enter Exam Marks


Search Exam
  • Please select the exam using Select Exam drop down.
  • Select the grade using the Select Grade drop down.
  • Then Select the year from the Year drop down. It is not mandatory.
  • Students and the subjects allocated for the Students will be Shown in the below table.


Enter Exam Marks
  • If you the exam is mentioned as the marks - you have to enter marks as Percentage
  • If the student is absent for a particular exam then mark as 'ab/AB'.
  • If the exam is optional then mark can be N/A.
  • Click on Save button to save the changes


Enter Exam Gradings
  • If the exam is mentioned as the gradings in the exam allocation page then you have to enter marks as gradings
  • If the student is absent for a particular exam then mark as 'ab/AB'.
  • If the exam is optional then mark can be N/A.
  • Click on Save button to save the changes
