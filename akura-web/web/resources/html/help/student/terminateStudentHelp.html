Terminate Student

Welcome to the Akura Management System. Using this page you terminate current students.



Past Student
  • All of the fields noted with an asterisk (*) are required.
    • Departure Date – Select the departure date of the student from calendar button.Calendar picker does not allow future dates.
    • Reason - It should be less than 251 characters.
  • Then tick the check box if the departure is complete clearance.
  • Click on “Save” button, then system asks "Are you sure you want to terminate this Student?".Click "OK" button. Then “Past Student” panel will close and send the email to the parent simultaneously by displaying the success message in green color at the top of the page as “Student has been successfully terminated & email has been sent to the parent”
  • If student successfully terminated but email is not sending because of server error/not giving parent email address: error should be displayed depend on the reason as “Student has been successfully terminated & unable to send the email due to server error or parent email address cannot be found”
  • Click the <Reset> button to reset the departure details form.
  • Clicking the “Cancel” button, the student terminated process going to stop and the “Past Student” panel closes.


Suspend Student
  • All of the fields noted with an asterisk (*) are required.
    • Disciplinary Actions Drop Down – Which will loads the student disciplinary actions of the current year.
    • From Date & To Date - Use calendar button to select a date.
      • ‘From Date’ Calendar picker loads only the current date of current year.
      • ‘To Date’ Calendar picker loads from current date of current year to days of one future year.
    • Click on “Save” button, the “Student Suspend” panel is closing and sending the email to the parent simultaneously by displaying the success message in green color at the top of the page as “Student has been successfully suspended & email has been sent to the parent”.
      • If student successfully suspended but email is not sending because of server error,
        • “Student has been successfully suspended
        • unable to send the email due to server error ”
      • If student successfully suspended but email is not sending because of parent email address is not exists,
        • “Student has been successfully suspended
        • unable to send the email due to parent email address cannot be found ”
      • If student successfully suspended but email is not sending because of parent details are not exists,
        • “Student has been successfully suspended
        • unable to send the email due to parent details are not exists ”
    • Clicking the “Cancel” button, the student suspended process going to stop and the “Student Suspend” panel closes.


Temporary Leave
  • All of the fields noted with an asterisk (*) are required.
    • From Date & To Date - Use calendar button to select a date.
      • ‘From Date’ Calendar picker loads only the current date of current year.
      • ‘To Date’ Calendar picker loads from current date of current year to days of 10 future years.
      • Reason - It should be less than 251 characters.
  • Then tick the check box if the termination is with complete clearance.
  • Click on “Save” button, then system asks "Are you sure you want to terminate this Student?".Click "OK" button.
    • If there are any incompete term marks, then system asks "Incomplete term marks will be deleted, are sure you want to remove this student?"
      • If user wants to proceed, click "OK" button and incomplete term records of the particular student will be deleted.
      • If user wants to manually complete the incomplete term marks, click "Cancel".
  • Then “Temporary Leave” panel will close and send the email to the parent simultaneously by displaying the success message in green color at the top of the page as “Student has been successfully terminated & email has been sent to the parent”
  • If student successfully terminated but email is not sending because of server error/not giving parent email address: error should be displayed depend on the reason as “Student has been successfully terminated & unable to send the email due to server error or parent email address cannot be found”
  • Clicking the “Cancel” button, the student terminated process going to stop and the “Temporary Leave” panel closes.




Close ‘Terminate Student’ Pop Up window
  • Clicking on ‘Close’ button before terminating the student , will close the 'Terminate Student' pop up window by redirecting to the ‘Search Student’ page.
  • Clicking on ‘Close’ button after successfully terminating the student will close the 'Terminate Student' pop up window and redirects to ‘Search Student’ page by viewing only terminated student link with the corresponding icon.
