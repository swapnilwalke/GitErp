Student Wise Attendance Report

Welcome to the Akura Management System. Using this page you can generate student swipe In/Out attendance view.

Student Attendance
  • Enter the following details:
    • Select Date From
    • Select Date To
    • Fields marked with * are mandatory to fill.
  • Click 'Search' button
  • If there is no reason, user can click on 'Add reason' link which will be redirect to the Student Absent tab.
