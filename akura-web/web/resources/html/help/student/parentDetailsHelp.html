Parent Details

Welcome to the Akura Management System. Using this page you can add/edit parent/guardian details of the selected student.

  • Add father/mother/guardian details
  • Edit father/mother/guardian details
  • Add donations
  • Edit donations
  • Delete donations


Parent Details - Add father/mother/guardian details
  • First Select the relationship (This selection is mandatory).
  • Enter the Parent details.
  • Labels displayed in the '*' are mandatory details.
  • Click on the Save button to save the newly added Parent details or click cancel button to clear the Changes.


Parent Details - Edit Parent/guardian Details
  • Parent details will be displayed for the particular student on the page.
  • Edit the Parent details.
  • Click on Save button to save edited Parent details or click cancel button to clear the Changes.


Parent Details - Add Donations
  • Enter the donation details.
  • Labels displayed in the '*' are mandatory details .
  • Labels displayed in the '**' are either one option is mandatory.
  • Click on the Save button to save added donation details or click cancel button to cancel the added details.


Parent Details - Edit Donations
  • First select the donation.
  • Click on the Edit mark in the selected record.
  • Then edit the details of the selected record.
  • Click Save button in donations to the school option for save edited details or click cancel button to cancel the edited details.


Parent Details - Delete Donations
  • First Select the donation.
  • Click Delete Icon of the selected record.
  • The confirmation box will display. Then click Ok button to delete selected record or cancel button to cancel the deletion.
