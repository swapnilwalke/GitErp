Teacher Subject Allocation

Welcome to the Akura Management System. Using this page you can allocate subjects and classes to a teacher.

  • Search Teacher Subject Allocation
  • Add New Grade Subject Allocation
  • Edit Teacher Subject Allocation
  • Delete Teacher Subject Allocation

Please click 'Add Icon' to Add a new teacher subject allocation, click 'Search Button' to view existing records, click 'Edit Icon' to edit selected record, click 'Delete Icon' to delete selected record.



Teacher Subject Allocation - Search Teacher Subject Allocation
  • Enter the following details:
    • Last name or part of the last name (Not mandatory)
    • Employee number (Not mandatory)
  • Click Search button to get a list of teacher suit the search criteria.


Teacher Subject Allocation - Add New Teacher Subject Allocation
  • Please click icon in the Teacher Subject Allocation Page.
  • Enter the following details:
    • Select Name of the Teacher (Note - In case if any teacher is not exist in drop down menu, you should add teacher from Add New Staff Member option in Staff Member Home page )
    • Select Grade Subject - The subject and the grade the teacher going to teach.(Note - In case if any grade subject is not exist in the drop down menu, you should add grade subject from Manage Grade Subject option in Reference Module)
    • tick on Classes check boxes - classes the teacher going to teach. (Note - In case if any class is not exist , you should add class from Manage Grade option in Reference Module)
  • Then Click Save button to save the newly entered details or Cancel to cancel newly entered details.
  • Fields marked with * are mandatory to fill.


Teacher Subject Allocation - Edit Teacher Subject Allocation
  • First Search the teacher.
  • Click on the icon of the Particular Teacher to edit teacher subject allocation details.
  • Then edit the details of the selected record using edit grade subject allocation option by editing teacher name, grade subjects and classes.
  • Then Click save button to save edited details or click cancel button to cancel the edited details.


Teacher Subject Allocation - Delete Teacher Subject Allocation
  • First Search the teacher.
  • Click on the icon of the selected record.
  • The confirmation box will display. Then click Ok button to delete selected record or cancel button to cancel the deletion.
