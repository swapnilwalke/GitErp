Staff Leave

Welcome to the Akura Management System. Using this page you can apply leave.

  • Staff Leave Types with the maximum no. of days, utilized days and available days for a relevant year will be displayed on the page.
  • Staff leaves,the user has taken for the relevant year will be displayed on the page
  • All of the fields noted with an asterisk (*) are required.
  • Before adding a staff leave make sure that the required reference data for Staff Leave Type has been entered to the system from the Reference Module.
  • <Home> link to redirect to the "Home" page.
  • <Staff Search> link to redirect to the "Staff Search" page.


Add New Leave
  • Please click add icon.
  • Enter the following details:
    • Type the reason for the leave
    • Select Staff leave Type (Note - In case if any Staff leave Type is not exist in drop down menu, you should add Staff leave Type from Add Staff leave Type option in Reference Module )
    • Select Start date and end date.
  • All the mandatory data that is given with red '' mark must be filled.
  • Click the 'Save' button or 'Cancel' button to cancel the applying the leave.


Edit Leave
  • Click on the edit icon of the relevant leave data.
  • Then edit data if you need
  • Click 'Save' button.


Delete Leave
  • Click on the 'Delete' icon.
  • Confirmation dialog box will be appeared.
  • Confirm deletion or cancel it.
