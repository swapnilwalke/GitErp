Class Teacher Allocation

Welcome to the Akura Management System. Using this page you can allocate teacher for the class.

  • Search Class Teacher allocation
  • Add Class Teacher allocation
  • Edit Class Teacher allocation
  • Delete Class Teacher allocation

Please click 'Add Icon' to Add a new Class Teacher allocation, click 'Search Button' to view existing records, click 'Edit Icon' to edit selected record, click 'Delete Icon' to delete selected record.




Class Teacher Allocation - Search Class Teacher Allocation
  • Enter the following details:
    • Last name or part of the last name (Not mandatory)
    • Class Grade(Not Mandatory)
    • Year(Default value is current year)
  • Click Search button to get a list of teacher suit the search criteria.


Class Teacher Allocation - Add Class Teacher Allocation
  • Enter the following details:
    • Please click icon in the Class Teacher Allocation Page.
    • Select Name of the Teacher (Note - In case if any teacher is not exist in drop down menu, you should add teacher from Add New Staff Member option in Staff Member Home page )
    • Select year
    • Select Grade Class (Note - In case if any Grade Class is not exist in drop down menu, you should visit reference module Manage Grade )
  • Then Click save button to Save the newly entered details or Cancel to cancel newly entered details.
  • Fields marked with * are mandatory to fill.


Class Teacher Allocation - Edit Class Teacher Allocation
  • First Search the teacher.
  • Click on the icon of the Particular Teacher to edit Class teacher allocation details.
  • Then edit the details of the selected record using edit grade class allocation option by editing teacher name, grade subjects and classes.
  • Then Click save button to save edited details or click cancel button to cancel the edited details.


Class Teacher Allocation - Delete Class Teacher Allocation
  • First Search the teacher.
  • Click icon of the selected record.
  • The confirmation box will display. Then click Ok button to delete selected record or cancel button to cancel the deletion.
