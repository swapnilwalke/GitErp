Extra Curricular

Welcome to the Akura Management System.Using this page you can manage staff extra curricular information for the selected staff.

  • If there are any existing extra curricular details, the data will be display on the page.
  • All of the fields noted with an asterisk (*) are required.
  • Before adding the Club or Society, Position, Sport and Seminar information make sure that the required reference data has been entered to the system from the Reference Module.
  • <Home> link to redirect to the "Home" page.
  • <Staff Search> link to redirect to the "Staff Search" page.


    • Clubs & Societies

      Add Clubs & Societies
    • To add clubs & societies details use the Add button and fill the required details, then click Save.


      • Edit Clubs & Societies
    • To edit information in this section, click the Edit button in the row section. Make changes to the appropriate fields, then click Save.


      • Delete Clubs & Societies
    • To delete information in these section, click the Delete button in the row section. Confirm the message to delete the record either using Ok or Cancel.
