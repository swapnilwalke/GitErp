Manage Grades

Welcome to the Akura Management System. The purpose of this section is to provide functionality to include grade related information in the system.


Functions provided under this module are


  • Grade Details
  • Add New Grade
  • Edit Grade
  • Delete Grade

Grade Details

This is the home page for the Grade section. It will display a list of Grades available in the system.


Add New Grade

  • Click on the add new Grade button, which would show the add new Grade page(Bottom of the current page).
  • Then enter the Grade name.
  • Tick classes from the check boxes.
  • Click on Save button to add the Grade.
  • Enter Grade name and click on the 'cancel' button. This will be re-directed to the Grade details page.

Edit Grade

  • Click on the edit Grade button, which would show the edit Grade page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the Grade name the classes.
  • Click on Save button to edit the Grade.
  • Edit Grade name and click on the 'cancel' button. This will be re-directed to the Grade details page.

Delete Grade

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Grade details page.
