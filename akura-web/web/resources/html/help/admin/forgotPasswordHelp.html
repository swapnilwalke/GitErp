Forgot Password

Welcome to the Akura Management System. The objective of this function is to help a user to get his password.


The user should enter the following details


  • Username - The username should be the one that user enter to login to the system. If the username is invalid then an error message will be displayed to the user to enter a valid user name.

Submit

Enter the username into the given field , Then click on submit button. On successful change of user password, a successful message will be displayed to the user and an email will be sent to the user's email. An error message will be displayed if email send failed.


Reset

Once the user enter the 'Reset' button. The username field will be set to empty.
