Manage Donation Type

Through this page you can manage reference data of Donation Type.

  • View already added Donation type list
  • Add New Donation type
  • Edit existing Donation type
  • Delete existing Donation type

When first time view the page it will be loaded with the existing list of Donation Types, click on icon to add new Donation type, click on icon to edit selected Donation Type and click on icon to delete selected Donation type.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



View Donation Type Details

This is the home page for the Donation Type section. It will display a list of Donation Types available in the system.


Add New Donation Type

  • Click on the add new Donation Type button, which would show the add new Donation Type page(Bottom of the current page).
  • Then enter the Donation type.
  • Click on Save button to add the Donation type.
  • After enter Donation type and click on the 'cancel' button. Then should be re-directed to the Donation Type details page.

Edit Donation Type

  • Click on the edit Donation Type button, which would show the edit Donation Type page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the Donation type.
  • Click on Save button to edit the Donation Type.
  • Edit Donation type and click on the 'cancel' button. Then should be re-directed to the Donation Type details page.

Delete Donation Type

  • Click on the delete button, which would show a popup with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Donation Type details page.
