Manage Staff Leave Types

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all general staff leaves related information in the system.


Functions provided under this module are


  • View Staff Leave Types Details
  • Add New Staff Leave Types
  • Edit Staff Leave Types
  • Delete Staff Leave Types

View Staff Leave Types Details

This is the home page for the general staff leave types in the school. It will display a list of staff leave types available in the system.


Add New Staff Leave Types

  • Click on Add New Staff Leave Type button, which would show the Add New Staff Leave Type panel (Bottom of the current page).
  • Then enter the Staff Leave Type and Maximum No of Staff Leaves
  • Click on Save button to add the staff leave type.
  • If clicks on the 'Cancel' button. This will be re-directed to the Staff Leave Types page by stopping the add process.

Edit Staff Leave Types

  • Click on the Edit Staff Leave Type button, which would show the Edit Staff Leave Type panel (Bottom of the current page)
  • The selected row will be highlighted in yellow color.
  • Then edit the staff Leave Type and Maximum No of Staff Leaves
  • Click on Save button.
  • If clicks on the 'Cancel' button. This will be re-directed to the Staff Leave Types page by stopping the edit process.

Delete Staff Leave Types

  • Click on Delete button, which would show a pop-up with the message Are you sure you want to delete this record? The selected row will be highlighted in yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Staff Leave Types page by stopping the delete process.
