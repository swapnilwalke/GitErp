Manage Employment Status

Through this page you can manage reference data of Employment Status.

  • View already added Employment Status list
  • Add Employment Status
  • Edit existing Employment status
  • Delete existing Employment status

When first time view the page it will loaded with the existing list of Employment Status, click on icon to add new Employment Status, click on icon to edit selected Employment Status and click on icon to delete selected Employment Status.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Employment Status
  • Click on the icon.
  • Then "Add New Employment Status" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Employment Status
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • fields marked with * are mandatory to fill.


Edit Employment Status
  • First select a Employment Status from the list to edit.
  • Click on the icon of the Particular Employment Status.
  • Then "Edit Employment Status" form will display on the Bottom of the current page. Edit the data, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Employment Status
  • First select a Employment Status from the list to delete.
  • Click on the icon of the Particular Employment Status.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
