Manage Staff Categories

Welcome to the Akura Management System. The purpose of this section is to provide functionality to include Staff Category related information in the system.


Functions provided under this module are


  • Staff Category Details
  • Add New Staff Category
  • Edit Staff Category
  • Delete Staff Category

Staff Category Details

This is the home page for the Staff Category section. It will display a list of Staff Categorys available in the system.


Add New Staff Category

  • Click on the add new Staff Category button, which would show the add new Staff Category page(Bottom of the current page).
  • Then enter the Staff Category name.
  • Select a staff type from radio buttons.
  • Click on Save button to add the Staff Category.
  • Enter Staff Category name and click on the 'cancel' button. This will be re-directed to the Staff Category details page.

Edit Staff Category

  • Click on the edit Staff Category button, which would show the edit Staff Category page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then you can edit the Staff Category and the staff type.
  • Click on Save button to edit the Staff Category.
  • Edit Staff Category name and click on the 'cancel' button. This will be re-directed to the Staff Category details page.

Delete Staff Category

  • Click on the delete button, which would show a popUp with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the staff Category details page.
