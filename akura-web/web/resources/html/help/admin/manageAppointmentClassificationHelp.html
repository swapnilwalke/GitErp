Manage Appointment Classification

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all Appointment Classification related information in the system. This section allows authenticated user to View, Add, Edit, Delete facilities for Appointment Classification.


Functions provided under this module are


  • View Appointment Classification
  • Add New Appointment Classification
  • Edit Appointment Classification
  • Delete Appointment Classification

View Appointment Classification

This is the home page for the Appointment Classification module. It will display a list of Appointment Classification , currently available in the system.


Add New Appointment Classification

  • Click on the 'Add New Appointment Classification' button, then you can view the Add New Appointment Classification panel(Bottom of the current page).
  • Then enter the new Appointment Classification name.
  • Click on 'Save' button to save the Appointment Classification.
  • Then you can view, entered Appointment Classification name in the list.
  • If click on the 'Cancel' button, Add New Appointment Classification panel will be disappeared and name will not be saved.

Edit Appointment Classification

  • Click on the 'Edit Appointment Classification' button, then you can view the Edit Appointment Classification panel (Bottom of the current page), and the selected row will be highlighted by yellow color.
  • Then edit the Appointment Classification name.
  • Click on 'Save' button to save the Appointment Classification.
  • If click on the 'Cancel' button, Edit Appointment Classification panel will be disappeared.

Delete Appointment Classification

  • Click on the 'Delete' button, then it will show a pop up with the message "Are you sure you want to delete this record?" and selected row will be highlighted by yellow color.
  • If click on 'OK' in the pop up message, the selected record will be deleted, whereas click 'Cancel' it will re-direct to the Appointment Classification details page.
