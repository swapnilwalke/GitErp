Manage Sports Age Groups

Through this page you can manage reference data for sport age groups.

  • View Sports Age Groups list
  • Add New Sports Age Group
  • Edit existing Sports Age Group
  • Delete existing Sports Age Group

The existing list of Sports Age Groups will be loaded for the first time, click on icon to add new Sports Age Group, click on icon to edit Sports Age Group and click on icon to delete Sports Age Group.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Sports Age Group
  • Click on the icon.
  • After that the "Add New Sports Age Group" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Sports Age Group
  • Click on 'Save' button to continue the process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Sports Age Group
  • First select a Sport Age Group from the list to edit.
  • Click on the icon of the Sports Age Group.
  • Then "Edit Sports Age Group" form will display on the Bottom of the current page. Edit the Sports Age Group field, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Sports Age Group
  • First select a Sport Age Group from the list to delete.
  • Click on the icon of the Sports Age Group.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
