Manage Warning Levels

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all general Warning Levels related information in the system.


Functions provided under this module are


  • View Warning Level Details
  • Add New Warning Level
  • Edit Warning Level
  • Delete Warning Level

View Warning Levels Details

This is the home page for the general warning levels in the school. It will display a list of warning levels available in the system.


Add New Warning Levels

  • Click on Add New Warning Level button, which would show the Add New Warning Level panel (Bottom of the current page).
  • Then enter the Warning Level and select Color
  • Click on Save button to add the warning level.
  • If clicks on the 'Cancel' button. This will be re-directed to the 'Warning Level' page by stopping the add process.

Edit Warning Levels

  • Click on the Edit Warning Level button, which would show the Edit Warning Level panel (Bottom of the current page)
  • The selected row will be highlighted in yellow color.
  • Then edit the Warning Level and Color
  • Click on Save button.
  • If clicks on the 'Cancel' button. This will be re-directed to the 'Warning Level' page by stopping the edit process.

Delete Warning Levels

  • Click on Delete button, which would show a pop-up with the message Are you sure you want to delete this record? The selected row will be highlighted in yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Warning Levels page by stopping the delete process.
