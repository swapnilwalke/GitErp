Manage Exams

Welcome to the Akura Management System. The purpose of this section is to provide functionality to include Exam related information in the system.

  • View already added Exam list
  • Add Exams
  • Edit existing Exams
  • Delete existing Exams

When adding exams it should be separated for different schemas. Eg. For A/L , A/L Science, A/L Commerce, A/L Arts


Then from the 'Exam Subjects' UI allocate subjects for that exams separately.


When first time view the page it will loaded with the existing list of Exams, click on icon to add new Exam, click on icon to edit selected Exam and click on icon to delete selected Exam.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Exam Procedure
  • Click on the icon.
  • Then "Add New Exam" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Exam
    • Select the Mark Type
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory to fill.


Edit Exam Procedure
  • First select a Exam from the list to edit.
  • Click on the icon of the Particular Exam.
  • Then "Edit Exam" form will display on the Bottom of the current page. Edit the data, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Exam Procedure
  • First select a Exam from the list to delete.
  • Click on the icon of the Particular Exam.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
