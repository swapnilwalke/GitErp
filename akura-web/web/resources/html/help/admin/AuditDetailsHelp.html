View Audit Details

Welcome to the Akura Management System. Using this page you can view audit details .

View Audit Details
  • Enter the following details:
    • Select user
    • Enter Business Function
    • Select From Date
    • Select Event Type
    • Enter Module
    • Select To Date
  • Click 'Search' button to get a list of audit details according to the search criteria.
  • Click 'View Details' icon to view more details about the specific audit record.
