Manage Assignment

Through this page you can manage Assignment for a grade - subject.

  • View already added Assignment Details
  • Add New Assignment Details
  • Edit existing Assignment Details
  • Delete existing Assignment Details

The existing list of Assignment Details for the grade subject will be loaded for the first time,Select the grade to view the List of subjects for that particular Assignment.
click on icon to add new Assignment of the selected grade-subject, click on icon to edit Assignment of the selected grade-subject and click on icon to delete Assignment of the selected grade-subject.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Assignment
  • Click on the icon.
  • Then "Add New Assignment" form will display on the Bottom of the current page.
  • Fill the following fields.
    • Select Grade
    • Select Subject
    • Submission Date, use to select a date.
    • Enter Assignment Name
    • Enter Assignment Description
    • Click on Assignment Marks Types
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Assignment
  • First select a Assignment from the list to edit.
  • Click on the icon of the Assignment.
  • Then "Edit Assignment" form will display on the Bottom of the current page. Edit the data grade, subject, assignment name, submission Date, description, assignment marks type and click on 'Save' button to continue the process or click 'Cancel' to stop editing.


Delete Assignment
  • First select a Assignment from the list to delete.
  • Click on the icon of the Assignment.
  • The confirmation box will display. Click 'OK' to continue the process or click 'Cancel' to stop delete.
