Manage Seminars

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all Seminar related information in the system. The storage of Seminar information is important when adding seminar to teacher.


Functions provided under this module are


  • View Seminar
  • Add New Seminar
  • Edit Seminar
  • Delete Seminar

View Seminar

It will display a list of Seminars available in the system.


Add New Seminar

  • Click on the add new Seminar button, which would show the add new Seminar page(Bottom of the current page).
  • Then enter the Seminar name.
  • Click on Save button to add the Professional Qualification.
  • After enter Seminar name and click on the 'cancel' button. This will be re-directed to the Seminar details page.

Edit Seminar

  • Click on the edit Seminar button, which would show the edit Seminar page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the Seminar name.
  • Click on Save button to edit the Seminar.
  • Edit Seminar and click on the 'cancel' button. This will be re-directed to the Seminar details page.

Delete Seminar

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Seminar details page.
