Manage Working Segment

Through this page you can manage reference data of Working Segments.

  • View already added Working Segments list
  • Add New Working Segment
  • Edit existing Working Segment
  • Delete existing Working Segment

The existing list of Working Segments will be loaded for the first time, click on icon to add new Working Segment, click on icon to edit Working Segment and click on icon to delete Working Segment.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Working Segment
  • Click on the icon.
  • After that the "Add New Working Segment" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Working Segment
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Working Segment
  • First select a Working Segment from the list to edit.
  • Click on the icon of the Particular Working Segment.
  • After that the "Edit Working Segment" form will display on the Bottom of the current page. Edit the data, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Working Segment
  • First select a Working Segment from the list to delete.
  • Click on the icon of the Particular Working Segment.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
