Manage Exam Subjects

Welcome to the Akura Management System. The purpose of this section is to provide functionality to include Exam Subject related information in the system.


Functions provided under this module are


  • Exam Subject Details
  • Add New Exam Subject
  • Edit Exam Subject
  • Delete Exam Subject

Exam Subject Details

This is the home page for the Exam Subject section. It will display a list of Exam Subject available in the system.


Add New Exam Subject

  • Click on the add new Exam Subject button, which would show the add new Exam Subject page(Bottom of the current page).
  • Select the Exam from the drop down list.
  • Subjects for the relevant grade of the exam will be displayed on the list box.
  • Select subjects of that exam from the list box.
  • Click on Save button to add the Exam Subject.
  • Enter Exam Subject name and click on the 'cancel' button. This will be re-directed to the Exam Subject details page.

Edit Exam Subject

  • Click on the edit Exam Subject button, which would show the edit Exam Subject page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Relevant exam will be displayed on the drop down.All other list will be disabled.
  • Then edit the Exam and the subjects.
  • Click on Save button to edit the Exam Subject.
  • Edit Exam Subject name and click on the 'cancel' button. This will be re-directed to the Exam Subject details page.

Delete Exam Subject

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Exam Subject details page.
