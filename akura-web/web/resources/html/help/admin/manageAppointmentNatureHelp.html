Manage Appointment Nature

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all appointment nature related information in the system. The storage of appointment nature related information is important to identify the appointment nature of the staff.


Functions provided under this module are


  • View appointment nature Details
  • Add New appointment nature
  • Edit appointment nature
  • Delete appointment nature

View appointment nature Details

This is the home page for the appointment nature section. It will display a list of appointment natures available in the system.


Add New appointment nature

  • Click on the add new appointment nature button, which would show the add new appointment nature page(Bottom of the current page).
  • Then enter the appointment nature description.
  • Click on Save button to add the appointment nature.
  • After enter appointment nature description and click on the 'cancel' button. This will be re-directed to the appointment nature details page.

Edit appointment nature

  • Click on the edit appointment nature button, which would show the edit appointment nature page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the appointment nature description.
  • Click on Save button to edit the appointment nature.
  • Edit appointment nature description and click on the 'cancel' button. Then will be re-directed to the appointment nature details page.

Delete appointment nature

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the appointment nature details page.
