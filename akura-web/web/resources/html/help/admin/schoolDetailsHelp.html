Welcome to the Akura Management System. Using this page you can setup the school details

  • Add/Update School details


Add/Update School details
  • Please enter the following required details:
    • School Name
    • Started Date
    • Country
    • Principal - if you don't have required data, you may have to visit staff module and add staff member as principle
    • Vice Principal - if you don't have required data, you may have to visit staff module and add staff member as vice principle
  • Then enter other non required details if you want
  • Click Save button to continue with the saving.
