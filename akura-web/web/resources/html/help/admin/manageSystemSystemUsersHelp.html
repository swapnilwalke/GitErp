Manage System Users

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all System users related information in the system.


Functions provided under this module are


  • Search System User
  • Add New User
  • Edit System User
  • Unlock User
  • Enable / Disable User
  • Delete User

Search System User

  • Enter a full name or partial name of a User Name.
  • Select the User Role from the 'User Role' drop down box.
  • Then click on search button.
  • Search result will populate for the particular entered User Name.


Add New System User

  • It will redirect to the System User Details page.

Edit System User

  • It will redirect to the System User detail page and it displays the selected details.

Unlock User

  • Click on the delete button, which would show a popUp with the message "Are you sure you want to Unlock this User?". The selected row will be highlighted by yellow color.
  • If click on 'OK' then it will unlock selected user.

Enable / Disable User

  • Click on the Enable/Disable button, which would show a popUp with the message "Are you sure you want to Enable or Disable this User?". The selected row will be highlighted by yellow color.
  • If click on 'OK' then it will enable or disable selected user.

Delete User

  • Click on the delete button, which would show a popUp with the message "Are you sure you want to delete this User?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Manage System Users details page.
