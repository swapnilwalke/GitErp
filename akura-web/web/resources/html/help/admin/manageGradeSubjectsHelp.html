Manage Grade Subjects

Welcome to the Akura Management System. The purpose of this section is to provide functionality to include Grade Subject related information in the system.


Functions provided under this module are


  • Grade Subject Details
  • Add New Grade Subject
  • Edit Grade Subject
  • Delete Grade Subject

Grade Subject Details

This is the home page for the Grade Subject section. It will display a list of Grade Subjects available in the system.


Add New Grade Subject

  • Click on the add new Grade Subject button, which would show the add new Grade Subject page(Bottom of the current page).
  • Select the Grade from the drop down list.
  • Tick subjects of that grade from the check boxes.
  • Click on Save button to add the Grade Subject.
  • Enter Grade Subject name and click on the 'cancel' button. This will be re-directed to the Grade Subject details page.

Edit Grade Subject

  • Click on the edit Grade Subject button, which would show the edit Grade Subject page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the Grade and the subjects.
  • Click on Save button to edit the Grade Subject.
  • Edit Grade Subject name and click on the 'cancel' button. This will be re-directed to the Grade Subject details page.

Delete Grade Subject

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the Grade Subject details page.
