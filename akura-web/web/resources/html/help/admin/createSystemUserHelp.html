Add/Update System User

Welcome to the Akura Management System. The objective of this function is to enable a administrative user to be added or updated system user details.


The user should enter the following details


  • User Role - Admin, Student, Teacher, Clerical Staff and Parent
  • Username - The username should be unique. If the username already exists then an error message will be displayed stating 'Sorry, this username already exists'
  • Password - Password must be at least 8 characters and Password cannot be Username or email address.
    Strength of the password will be shown in a color bar. Color of the color bar will depend on the strength of the Password.
    To secure your password:
    • Use both letters and numbers
    • Add special characters (such as $, &, ?)
    • Mix capital and lowercase letters
  • Confirm Password
  • Admission No/Registration No - Admission Number should be filled when create a user with student role. Registration Number should be filled when create a user with teacher role. It is not necessary to filled when create an administrator user.
  • First Name
  • Last Name
  • Email Address - The email address should be in the correct format. If an incorrect format is provided then the error message will be displayed as 'Incorrect email format! Please enter a valid email address', next to the email address field
  • Note: When updating an User detail, user can not update 'User Role' and 'User Name'. If user role is Teacher, Clarical Staff, Student or Parent they can not update their identification Numbers such as For student Admission No. etc.

Save

Enter the mandatory field which is marked by * , Then click on Save button. On successful user creation, The Add New System User page will be displayed with the message 'System User Created Successfully!' And an email notification will be sent to the newly created user, containing user's newly created username and password


Reset

Enter all required information and then click the 'Reset' button. All the information entered previously will be reset to empty values.


Cancel

Enter all required information and click on the 'Cancel' button. This will be re-directed to the previous page
