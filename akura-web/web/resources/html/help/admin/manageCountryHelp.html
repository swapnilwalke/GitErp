Manage countries

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all country related information in the system. The storage of country related information is important to identify the country in which the school is located


Functions provided under this module are


  • Search country
  • View country Details
  • Add New country
  • Edit country
  • Delete country

Search country

  • Enter a full name or partial name of a country.
  • Then click on search button.
  • Search result will populate for the particular entered country.

View country Details

This is the home page for the country section. It will display a list of countries available in the system.


Add New country

  • Click on the add new country button, which would show the add new country page(Bottom of the current page).
  • Then enter the country name.
  • Click on Save button to add the country.
  • After enter country name and click on the 'cancel' button. This will be re-directed to the country details page.

Edit country

  • Click on the edit country button, which would show the edit country page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Click on Save button to edit the country.
  • Edit country name and click on the 'cancel' button. This will be re-directed to the country details page.

Delete country

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the country details page.
