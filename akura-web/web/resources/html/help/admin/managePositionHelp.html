Manage Membership Positions

Through this page you can manage reference data of Membership Positions.

  • View already added Membership Positions list
  • Add New Membership Positions
  • Edit existing Membership Positions
  • Delete existing Membership Positions

The existing list of Membership Positions will be loaded for the first time, click on icon to add new Membership Positions, click on icon to edit Membership Positions and click on icon to delete Membership Positions.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Membership Positions
  • Click on the icon.
  • After that the "Add New Membership Positions" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Membership Positions
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Membership Positions
  • First select a Membership Position from the list to edit.
  • Click on the icon of the Membership Position.
  • After that the "Edit Membership Position" form will display on the Bottom of the current page. Edit the Membership Positions field, and click on 'Save' button to continue the process or click 'Cancel' to stop editing.


Delete Membership Positions
  • First select a Membership Position from the list to delete.
  • Click on the icon of the Membership Position.
  • The confirmation box will display. Click 'OK' to continue the process or click 'Cancel' to stop delete.
