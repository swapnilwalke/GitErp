Change Password

Welcome to the Akura Management System. The objective of this function is to help a user to change his password.


The user should enter the following details


  • Old Password - The old password should be the one that user enter to login to the system. If the password is invalid then an error message will be displayed to the user to enter a valid password.
  • New Password - This should the password that you want to use in the future.
  • Confirm Password - This is just to confirm your new password. Enter your new password to confirm.

Next

Enter the required details into given fields , Then click on next button. On successful change of user password, user will be logged into the system and an email will be sent to the user's email.
