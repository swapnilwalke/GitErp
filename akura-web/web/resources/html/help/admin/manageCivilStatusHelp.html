Manage Civil Status

Welcome to the Akura Management System. The purpose of this section is to provide functionality to handle all Civil Status related information in the system. The storage of Civil Status related information is important to identify the Civil Status of the users .


Functions provided under this module are


  • View Civil Status Details
  • Add New Civil Status
  • Edit Civil Status
  • Delete Civil Status

View Civil Status Details

This is the home page for the Civil Status section. It will display a list of Civil Status available in the system.


Add New Civil Status

  • Click on the add new Civil Status button, which would show the add new Civil Status page(Bottom of the current page).
  • Then enter the Civil Status name.
  • Click on Save button to add the Civil Status.
  • After enter Civil Status name and click on the 'cancel' button. This will be re-directed to the Civil Status details page.

Edit Civil Status

  • Click on the edit Civil Status button, which would show the edit Civil Status page(Bottom of the current page). The selected row will be highlighted by yellow color.
  • Then edit the Civil Status name.
  • Click on Save button to edit the Civil Status.
  • Edit Civil Status name and click on the 'cancel' button. This will be re-directed to the Civil Status details page.

Delete civil status

  • Click on the delete button, which would show a pop up with the message "Are you sure you want to delete this record?". The selected row will be highlighted by yellow color.
  • If click on 'OK' the selected record will get delete, if click 'Cancel' it will re-directed to the civil status details page.
