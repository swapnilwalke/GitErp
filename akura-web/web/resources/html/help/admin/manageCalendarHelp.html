Manage Holiday

Through this page you can manage Holidays of a year.

  • View already added Holiday Details
  • Add New Holiday Details
  • Edit existing Holiday Details
  • Delete existing Holiday Details

The existing list of Calendar Details for the current year will be loaded for the first time,Select the year to view the List of Holidays for that particular year.
click on icon to add new Holiday of the selected year, click on icon to edit Holiday of the selected year and click on icon to delete Holiday of the selected year.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Holiday
  • Click on the icon.
  • Then "Add New Holiday" form will display on the Bottom of the current page.
  • Fill the following fields.
    • Start Date, use to select a date.
    • End Date, use to select a date.
    • Enter Reason for Holiday
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Holiday
  • First select a Holiday from the list to edit.
  • Click on the icon of the Holiday.
  • Then "Edit Holiday" form will display on the Bottom of the current page. Edit the data start Date, End Date and/or reason and click on 'Save' button to continue the process or click 'Cancel' to stop editing.


Delete Holiday
  • First select a Holiday from the list to delete.
  • Click on the icon of the Holiday.
  • The confirmation box will display. Click 'OK' to continue the process or click 'Cancel' to stop delete.
