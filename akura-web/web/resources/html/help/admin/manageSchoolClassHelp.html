Manage Classes

Through this page you can manage reference data of School Classes.

  • View already added School Class list
  • Add New class
  • Edit existing classes
  • Delete existing classes

When first time view the page it will be loaded with the existing list of Classes, click on icon to add new Class, click on icon to edit selected Class and click on icon to delete selected Class.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Class Procedure
  • Click on the icon.
  • Then "Add New Class" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Class
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • fields marked with * are mandatory to fill.


Edit Class Procedure
  • First select a Class from the list to edit.
  • Click on the icon of the Particular Class.
  • Then "Edit Class" form will display on the Bottom of the current page. Edit the data, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Class Procedure
  • First select a Class from the list to delete.
  • Click on the icon of the Particular Class.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
