Manage Sports Categories

Through this page you can manage reference data for sports Categories.

  • View Sport Category list
  • Add New Sport Category
  • Edit existing Sport Category
  • Delete existing Sport Category

The existing list of Sport Categories will be loaded for the first time, click on icon to add new sport Category, click on icon to edit Sport Category and click on icon to delete Sport Category.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Sport Category
  • Click on the icon.
  • After that the "Add New Sport Category" form will display on the Bottom of the current page.
  • Select the following field(s).
    • Name of the sport
    • Name of the sport age group
  • Click on 'Save' button to continue the process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Sport Category
  • First select a Sport Category from the list to edit.
  • Click on the icon of the selected sport Category.
  • Then "Edit Sport Category" form will display on the Bottom of the current page. Edit the Sport field and Sport Age Group field, and click on 'Save' button to continue the process or click 'Cancel' to stop editing.


Delete Sport Category
  • First select a Sport Category from the list to delete.
  • Click on the icon of the selected Sport Category.
  • The confirmation box will display. Click 'OK' to continue the process or click 'Cancel' to stop delete.
