Manage Special Events

Through this page you can manage reference data for Special Events.

  • View Special Events and Participants list
  • Add New Special Events and Participants
  • Edit existing Special Events and Participants
  • Delete existing Special Events

The existing list of Special Events will be loaded for the first time, click on icon to add new Special Event, click on icon to edit Special Events and click on icon to delete Special Events.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Special Events and Participants List
  • Click on the icon.
  • After that the "Add New Special Events and Participants" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Name of the Special Event
    • Description of the Special Event
    • Date of the Special Event
    • Participant Category of the Special Event
    • Once Participant Category is selected, Eligible Participants list will be populate.
    • select Participant and add it to Selected Participants list.
  • Click on 'Save' button to continue the process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory and should be filled.


Edit Special Event and Participants List
  • First select a Special Event from the list to edit.
  • Click on the icon of the selected Special Event.
  • Then "Edit Special Event and Participants" form will display on the Bottom of the current page. Edit the fields, and click on 'Save' button to continue the process or click 'Cancel' to stop editing.


Delete Special Event
  • First select a Special Event from the list to delete.
  • Click on the icon of the selected Special Event.
  • The confirmation box will display. Click 'OK' to continue the process or click 'Cancel' to stop delete.
