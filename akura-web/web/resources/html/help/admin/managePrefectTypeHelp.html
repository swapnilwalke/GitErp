Add Prefect Types

Through this page you can manage reference data of Prefect Types.

  • View already added Prefect Type list
  • Add Prefect Types
  • Edit existing prefect types
  • Delete existing prefect types

When first time view the page it will loaded with the existing list of Prefect Types, click on icon to add new Prefect Type, click on icon to edit selected Prefect Type and click on icon to delete selected Prefect Type.


  • <Home> link to redirect to "Home" page.
  • <Reference Module> link to redirect to "Reference Module" page.



Add New Prefect Type Procedure
  • Click on the icon.
  • Then "Add New Prefect Type" form will display on the Bottom of the current page.
  • Fill the following field(s).
    • Class
  • Click on 'Save' button to continue add process or click 'Cancel' to stop adding.
  • Fields marked with * are mandatory to fill.


Edit Prefect Type Procedure
  • First select a Prefect Type from the list to edit.
  • Click on the icon of the Particular Prefect Type.
  • Then "Edit Prefect Type" form will display on the Bottom of the current page. Edit the data, and click on 'Save' button to continue edit process or click 'Cancel' to stop editing.


Delete Prefect Type Procedure
  • First select a Prefect Type from the list to delete.
  • Click on the icon of the Particular Prefect Type.
  • The confirmation box will display. Click 'OK' to continue delete process or click 'Cancel' to stop delete.
