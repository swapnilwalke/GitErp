Answer Security Questions

Welcome to the Akura Management System. The objective of this function is to help a user to answer security questions.


The user should enter the following details


  • Answer One - User should be able to provide the answer for the security question one that he/she selected before.
  • Answer Two - User should be able to provide the answer for the security question two that he/she selected before.

Next

Enter the required details into given fields , Then click on next button. If your answers are incorrect, an error message will be displayed.
