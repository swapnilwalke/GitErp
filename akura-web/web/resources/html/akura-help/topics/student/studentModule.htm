Student Module

The Student module allows you to maintain student details.

  • Search for a student and perform any of the following tasks:
    • Edit student details.
    • Manage religious activities.
    • Manage academic information.
    • Manage co-curricular details.
    • Manage discipline.
    • Manage student absence. 
    • Manage parent details.
    • Manage attendance.
    • Message the parent/guardian.
  • Assign students to a class.
  • Assign subjects to students.
  • Enter student examination marks.
  • Enter student marks.
  • Enter student assignment marks.
