Student Assignment Marks

Navigation: Select Student Module->Student Assignment Mark

Using this page, you can enter assignment marks for students. Entering assignment marks is a two-step process:

  • Search for students.
  • Enter assignment marks.
Note: Suspended student marks are highlighted.

Search for Students

  1. Select the class from the Select Class drop down. All the assignments for the selected class will be shown in the Select Assignment drop down.
  2. Select the assignment from the Select Assignment drop down.
  3. Click the Search button to view the search results.

Enter Assignment Marks or Grades

The assignment marks must be entered as a percentage or grade depending on how the Assignment Mark Type is specified in the Reference module.

  1. To enter the marks, do one of the following:
    • If the Assignment Mark Type is Marks, enter the marks as a percentage.
    • If the Assignment Mark Type is Grades, enter the marks as grades (A, B, C, etc.).
  2. Click Save to save the marks or click Cancel to cancel the operation.
