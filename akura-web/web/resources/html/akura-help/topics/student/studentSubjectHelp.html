Assign Subjects to Students

Navigation: Select Student Module->Student Subjects.

Using this page, you can assign subjects to students. Assigning subjects to students is a two-step process:

  • Search for a class.
  • Assign subjects to the students in the class.

Notes:

  • You must assign students to a class before you can assign subjects to them.
  • Core subjects are automatically assigned to the students.
  • Suspended students or students on temporary leave are not listed under classes.

Search for Class

  1. Select the class using the Select Class drop down menu.
  2. Select the year using the Select Year drop down menu.
  3. Click the Search Class button. The students and subjects for the selected class will be displayed along with any current subject assignments.

Assign Subjects to Students

  1. To assign subjects, do one of the following:
    • To assign subject(s) to a student, select the relevant subject check box(es) next to the student's name.
    • To remove subject assignments, deselect the relevant subject check box(es) next to the student's name.
    • To assign all the subjects to all the students, click the Check All button.
  2. Click Save to assign the subjects, or click Cancel to cancel the operation.
