Activate Suspended Student

Navigation: Select Student Module->Search Student->Activate Suspended Student

Using this page, you can reinstate the student career of a suspended student. 

Suspended Student History Details

The page displays the suspension history of a student, which includes the reason for the suspension, duration and the disciplinary action taken.

Activate Suspended Student

Note: Fields marked with * are mandatory.
  1. Select the student status as Suspended from the dropdown menu.
  2. Click the Search button.
  3. Click the Activate Suspended Student  icon of the relevant student.
  4. The Activate Suspended Student Service window pops up.
  5. Click the Activate Suspended Student button. 
  6. Select the date of activation using the calendar icon.
  7. Provide a Curtailed/Extended reason.
  8. Click the Activate button. A confirmation message will be displayed.
  9. Click OK to confirm the activation. Click Cancel to cancel the activation.
