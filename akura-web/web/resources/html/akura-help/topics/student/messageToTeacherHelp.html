Send a Message to a Teacher

Navigation: Select Student Module->Search Student->Edit->Message Board
Using this page, you can send messages to a teacher via email.

Send Message

  1. Select a teacher (subject or class) from the drop down.
  2. Enter the subject of the message in the Message Subject text box.
  3. Type the message.
  4. Click the Send button.
