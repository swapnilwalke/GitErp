Assign Students to a Class

Navigation: Select Student Module->Student Class

Using this page, you can assign students to a class. Assigning students to a class is a two step process:

  • Search for students.
  • Assign the students to a class.

Search for Students

  1. Select the grade from the Select Grade drop down menu. For new students, select 'New Students' from the drop down menu.
  2. Select the year from the Select Year drop down menu.
  3. Click the Search Students button. Students that matches the given criteria will be listed in the Students in () list box

Assign Students to a Class

  1. Select the students from the Students in () list box.
  2. Click the right arrow button to add the students to the Selected Students list box. (Note that you can remove students from the Selected Students list by clicking the left arrow button.)
  3. Select the class from the Assign to Class drop down menu.
  4. Select the year from the Assign to Year drop down menu.
  5. Click Save to assign the students to the class and year or click Cancel to cancel the operation.
Notes:
  • Suspended students or students on temporary leave are not listed under grades.
  • To assign new students who are not assigned to a class,  select "New Students" from the select grade menu and the joined year respectively.
