Delete a Student

Navigation: Select Student Module-> Student Search ->Delete

Using this page, you can delete a student in the the system.

Delete a Student

  1. Use the search feature to to find the relevant student.
  2. Click the Delete icon of the relevant student to remove the student from the system.  
  3. Click the OK button to confirm the deletion.
