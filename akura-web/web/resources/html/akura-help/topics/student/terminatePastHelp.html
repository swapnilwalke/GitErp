Student Departure

Navigation: Select Student Module->Search Student->Terminate Student. 

Using this page, you can terminate the school tenure of a student. 

Student Departure

  1. Use the Search feature to find the relevant student. 
  2. Click the Terminate Student button. Terminate Student prompt will be opened. 
  3. Select the Terminate Criteria as Past. 
  4. Select the Departure Date. 
  5. Provide a reason
  6. Check the Complete Clearence check box.
  7. Click the Save button to confirm the Terminate operation. Click Reset to reset the fields.
