Edit Students

Navigation: Select Student Module->Search Student->Edit Student

Using this page, you can add/edit various student information. 

  1. Click the Student Module.
  2. Search for the relevant student or click search to list all the students in the system. 
  3. Click the name of the student or click the Edit Student icon.
