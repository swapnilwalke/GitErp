Student Progress Summary

Navigation: Select Student Module->Search Student->Edit->Message Board

Using this page, you can generate a student's progress summary. You can save the report on your computer and mail it to the student's parent or guardian as an attachment.

View Student Progress Summary

  1. Enter a comment for the relevant student.
  2. Click the Preview Report button.

     The Student Progress Summary shows progress in the following areas:

    • Line chart of the student's marks for all terms for the current year categorized by the subjects
    • Grades for the relevant subjects for the sub-terms
    • Attendance data

Email the report

    1. Enter a comment for the relevant student.
    2. Click the Preview Report button.

    The Student Progress Summary shows progress in the following areas:

      • Line chart of the student's marks for all terms for the current year categorized by the subjects
      • Grades for the relevant subjects for the sub-terms
      • Attendance data
    1. Click the Send Email button to send the Student Progress Summary to the parent or guardian.
    Note: Parent/Guardian e-mail address should be defined to send an e-mail.
