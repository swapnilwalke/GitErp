Attendance Reporting System

Navigation: Select Student Module->Search Student->Edit->Attendance

Using this page, you can view attendance (swipe in/out) details for a selected student.

Notes: 

  • If there are any attendance records, the data will be displayed in the page.
  • Make sure that the attendance for the student has been entered to the system through the Attendance module.
  • Fields marked with * are mandatory.
  • The Attendance progress bar is used to display a summary of the student's attendance details for the selected year.
  • The Religious Activities progress bar is used to display a summary of the student's Religious Activities for the selected year.
  • The Academic Life progress bar is used to display a summary of the student's academic life for the selected year.
  • Clicking the Home button will take you to the Home page. 
  • Clicking the student Search button will take you to the Student Search page.

View Student Attendance

  1. Select the date range to view attendance.
  2. Click the Search button.
  3. For days absent, you can add a reason for absence by clicking the Add Reason link. The Absent page is displayed.
  4. Enter a reason for absence and select the date range.
  5. Click Save.
