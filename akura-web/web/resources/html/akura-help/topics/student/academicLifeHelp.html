Academic Life

Navigation: Select Student Module->Search Student->Edit->Academic Life

Using this page, you can manage academic life information of students.

Notes:

  • If there are any existing academic life details, the data will be displayed in the page.
  • Fields marked with * are mandatory.
  • Before adding the prefect information and scholarship information, make sure that the required reference data has been entered to the system through the Reference module.
  • The Attendance progress bar displays a summary of the student's attendance details for the selected year.
  • The Religious Activities progress bar displays a summary of the student's Religious Activities for the selected year.
  • The Academic Life progress bar displays a summary of the student's academic life for the selected year.
  • Clicking the Home button will take you to the home page.
  • Clicking the student Search button will take you to the Student Search page.

Subject Marks

In this section, you can view the following:

  • The student's Term Marks with the total marks and average marks
  • The student's Sub-term Marks by clicking on a subject link
  • The student's Assignment Marks by clicking on the icon next to the subject
  • The student's Exam Marks by clicking the icon or the View Exam Marks Details link

Prefect Details

Add Prefect Details

  1. To add prefect details, click the Add icon and fill in the required details.
  2. Click Save to save the details or click Cancel to cancel the add operation.

Edit Prefect Details

  1. To edit prefect details, click the Edit icon in the row section.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Prefect Details

  1. To delete prefect details, click the Delete icon in the row section.
  2. A confirmation message is displayed. Click OK to delete the prefect details or click Cancel to cancel the deletion.
