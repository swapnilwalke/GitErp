Student Temporary Leave

Navigation: Select Student Module->Search Student->Terminate Student

Using this page, you can grant temporary leave for a selected student.

Temporary Leave

  1. Use the Search feature to find the relevant student. 
  2. Click the Terminate Student button. Terminate Student prompt will be opened. 
  3. Select the Terminate Criteria as Temporary Leave. 
  4. Select the duration from the Calendar buttons.  
  5. Provide a reason.
  6. Check the Complete Clearence check box. 
  7. Click the Save button to confirm the operation.
