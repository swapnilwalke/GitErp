Terminate Students

Navigation: Select Student Module->Search Student->Terminate Student

Using this page, you can terminate students, grant temporary leave for students and suspend students in the system.  

Note: Fields marked with * are mandatory.

Student Departure

  1. Use the Search feature to find the relevant student. 
  2. Click the  button. The Terminate Student prompt is opened.
  3. Select the Terminate Criteria as Past. 
  4. Select the Departure Date. 
  5. Provide a reason.
  6. Check the Complete Clearance check box.
  7. Click the Save button to confirm the Terminate operation or click Reset to reset the fields. 

Temporary Leave

  1. Use the Search feature to find the relevant student. 
  2. Click the  button. The Terminate Student prompt is opened.
  3. Select the Terminate Criteria as Temporary Leave. 
  4. Select the duration from the Calendar buttons.  
  5. Provide a reason.
  6. Check the Complete Clearance check box. 
  7. Click the Save button to confirm the operation. 

Suspend Students

  1. Use the Search feature to find the relevant student. 
  2. Click the  button. The Terminate Student prompt is opened.
  3. Select the Terminate Criteria as Suspend. 
  4. Select the disciplinary action from the drop down menu.
  5. Select the duration from the Calendar buttons.  
  6. Provide a description. 
  7. Click the Save button to confirm the suspend operation.
