Student Exam Marks

Navigation: Select Student Module->Student Exam Marks Entry

Using this page, you can enter exam marks for the students. Entering exam marks is a two-step process:

  • Search for the exam and grade.
  • Enter exam marks or grades.

Note: When adding exams, add separate exams for different schemes. E.g. For Advanced Level, there should be separate exams for A/L Science, A/L Commerce and A/L Arts. Then from the Manage Exam Subjects UI, allocate subjects for each exam.

Search for Exam

  1. Select the exam from the Select Exam drop down.
  2. Select the grade from the Select Grade drop down.
  3. Select the year from the Year drop down.(not mandatory)
  4. Click the Search Exam button. The students and subjects for the selected exam will be displayed.

Enter Exam Marks or Grades

  1. Enter the exam index number.
  2. To enter the marks, do one of the following:
    • If the Exam Mark Type is "Marks", enter the marks as a percentage.
    • If the Exam Mark Type is "Grades", enter the marks as grades (A, B, C, etc.).
  3. Click the Save button to save the marks or click Cancel to cancel the operation.
Notes:
  • If a student is absent for an exam, the mark should be "ab/AB". If the exam is optional, then the mark can be "N/A".
  • The exam marks must be entered as a percentage or grade depending on how the Exam Mark Type is specified in the Reference module.
  • Suspended student records are highlighted.
