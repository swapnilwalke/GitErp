Search Student

Navigation: Select Student Module->Search Student

Using this page, you can search for a student using a specific criterion. There are two types of search:

  • Search - You can perform a simple search based on a student's last name, admission number or grade.
  • Advanced Search: You can perform a search based on additional criteria such as class, subject, sport, etc.

Note: Clicking the Add New Student button will take you to the Add Student Details page.

Perform a Search

  1. Provide one or more of the following search criteria:
    • Last name
    • Admission number
    • Grade
    • Student Status
  2. Click the Search button to view the search results.
  3. You can edit student details by clicking on a student name.

Note: 

  • When the Search button is clicked without setting any search criteria, system lists all the students in the system. 
  • Last name field can be a part of the last name.
  • Fields marked with * are mandatory.
  • To search for new students, select the Grade as New Students from the dropdown menu. 
  • No icons are displayed in front of past students. 
  • New students are labeled as "New Students" under the Student Class column.

Perform an Advanced Search

  1. Provide one or more of the following search criteria:
    • Class
    • Sport
    • Subject
    • Prefect Type
    • Parent's Profession
  2. Click the Search button to view the search results.
  3. You can edit student details by clicking on a student name or by clicking the Edit Student button.
