Suspend Students

Navigation: Select Student Module->Search Student->Terminate Student. 

Using this page, you can suspend students as a desciplinary action. 

Suspend Students

  1. Use the Search feature to find the relevant student. 
  2. Click the Terminate Student button. Terminate Student prompt will be opened. 
  3. Select the Terminate Criteria as Suspend. 
  4. Select the Disciplinary action from the drop down menu.
  5. Select the duration from the Calendar buttons.  
  6. Provide a description. 
  7. Click the Save button to confirm the suspend operation.
