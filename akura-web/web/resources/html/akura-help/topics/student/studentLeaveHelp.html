Student Leave

Navigation: Select Student Module->Search Student->Edit->Absent

Using this page, you can apply, edit and delete student leave.

Notes: 

  • Existing leave/absence records will be listed on the Absent tab. 
  • Fields marked with * are mandatory.
  • Before adding values make sure that the required reference data has been entered to the system through the Reference module.
  • The Attendance progress bar is used to display a summary of the student's attendance details for the selected year.
  • The Religious Activities progress bar is used to display a summary of the student's Religious Activities for the selected year.
  • The Academic Life progress bar is used to display a summary of the student's academic life for the selected year.
  • Clicking the Home button will take you to the Home page.

Apply for Student Leave

  1. Click the Add icon.
  2. Add the description and select the duration using the calendar icons.
  3. Click Save to save the details or click Cancel to cancel the add operation.

Edit Student Leave

  1. To edit student leave details, click the Edit icon in the row section.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Leave

  1. To delete student leave, click the Delete icon in the row section.
  2. A confirmation message is displayed. Click OK to delete the leave or click Cancel to cancel the deletion.
