Manage Cities

Navigation: Select Reference Module->Manage Cities

Using this page, you can search for a city and then view, add, edit and delete cities in the system.

View City Details

This is the home page for the City section. It displays a list of cities available in the system.

Search City

  1. Enter the full name or partial name of a city.
  2. Click the Search button to view the search results. 

Add New City

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the name of the city.
  3. Select a district from the drop down menu.
  4. Click the Save button to add the city.
  5. Click the Cancel button to go to the City Details page.

Edit City

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the name of the city and change the district as required.
  3. Click the Save button to save the changes.
  4. Click the Cancel button to go to the City Details page.

Delete City

Note: Data shared by other modules cannot be deleted.
  1. Click the button. The selected row is highlighted in yellow. 
  2. Click OK to delete the selected record or click Cancel to go to the City Details page.
