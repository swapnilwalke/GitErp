Staff

Reference module functions are divided in to six categories for the ease of your reference. "Staff" category lists the following staff member related functions in the system.

  • Manage Sections
  • Manage Civil Status
  • Manage Staff Category Information
  • Manage Appointment Nature
  • Manage Appointment Classification
  • Manage Staff Service Category
  • Manage Educational Qualifications
  • Manage Professional Qualifications
  • Manage Staff Leave Types
