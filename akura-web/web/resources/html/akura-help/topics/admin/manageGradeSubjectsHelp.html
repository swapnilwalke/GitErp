Manage Grade Subjects

Navigation: Select Reference Module->Manage Grade Subjects

Using this page, you can view, add, edit and delete Grade Subject related information in the system.

View Grade Subject Details

This is the home page for the Grade Subject section. It displays a list of grade subjects available in the system.

Add New Grade Subject

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Select the grade from the drop down list.
  3. Select available subjects from the Subject list box. 
  4. If necessary, select the Optional check box. 
  5. Enter Maximum Marks for the subjects.
  6. Click the Save button to add the Grade Subject.
  7. Click the Cancel button to go to the Grade Subject Details page.

Edit Grade Subject

  1. Click the button. The selected row is highlighted in yellow. 
  2. Add or remove subjects. 
  3. If necessary, select the Optional check box. 
  4. Edit Maximum Marks for the subjects.
  5. Click the Save button to save the changes. 
  6. Click the Cancel button to cancel the changes. 
Note: Subjects cannot be moved after student marks have been entered for that grade. 

Delete Grade Subject

  1. Click the button. The selected row is highlighted in yellow. 
  2. Click the OK button to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
