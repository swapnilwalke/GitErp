Answer Security Questions

Navigation: Answer Security Question prompt appears in the initial login. 

Using this page, you can reset your password by answering the security questions. 

  1. To change the password, enter the following details:
    • Answer One - Enter the answer for the first security question.
    • Answer Two - Enter the answer for the second security question.
  2. Click the Next button. If your answers are incorrect, an error message will be displayed.
