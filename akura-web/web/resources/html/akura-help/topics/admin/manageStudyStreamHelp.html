Manage Streams

Navigation: Select Reference Module->Manage Study Streams

Using this page, you can search and then view, add, edit and delete study stream details in the system.

View Stream Details

This is the home page for the Stream section. It displays a list of study streams available in the system.

Search Stream

  1. Enter the full name or partial name of a study stream.
  2. Click the Search button to get the search results.

Add New Stream

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the study stream name.
  3. Click the Save button to add the study stream or click Cancel to cancel the add operation. 

Edit Stream

  1. Click the button. The selected study stream is highlighted in yellow.
  2. Edit the study stream name.
  3. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Stream

  1. Click the button of the relevant study stream. The selected study stream is highlighted in yellow.
  2. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
