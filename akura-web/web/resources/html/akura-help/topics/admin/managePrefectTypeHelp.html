Manage Prefect Types

Navigation: Select Reference Module->Manage Prefect Types

Using this page, you can view, add, edit and delete the prefect types in the system.

View Prefect Types

This is the home page of the Prefect Types section. It displays a list of prefect types available in the system.

Add New Prefect Type Procedure

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Enter the prefect type name.
  3. Click the Save button to save the details or click Cancel to cancel the add operation.

Edit Prefect Type Procedure

  1. Select a prefect type from the list to edit. The selected row is highlighted in yellow. 
  2. Click the icon of the relevant prefect type.
  3. Edit the name of the prefect type.
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Prefect Type Procedure

Note: Data shared by other modules cannot be deleted.
  1. Select a prefect type from the list to delete. The selected row is highlighted in yellow. 
  2. Click the icon of the relevant prefect type.
  3. A confirmation message is displayed. 
  4. Click the OK button to confirm the deletion or click Cancel to cancel the deletion.
