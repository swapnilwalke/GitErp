Manage Nationalities

Navigation: Select Reference Module->Manage Nationality

Using this page, you can view, add, edit and delete nationality related information in the system. 

View Nationality Details

This is the home page of the Nationality section. It displays a list of nationalities available in the system.

Add New Nationality

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the nationality.
  3. Click the Save button to add the nationality or click Cancel to cancel the operation.

Edit Nationality

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the nationality.
  3. Click the Save button to save the change or click Cancel to cancel the change.

Delete nationality

Note: Data shared by other modules cannot be deleted.
  1. Click the button. A confirmation message is displayed. The selected row is highlighted in yellow. 
  2. Click OK to confirm the deletion or click Cancel to cancel the deletion.
