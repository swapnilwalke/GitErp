Forgot Username

Navigation: Login->Forgot Username
Using this page, you can retrieve your username if you have forgotten it. 

  1. To retrieve the username, enter the email address that you entered when you created the user account.
  2. Click the Submit button. If the email address is invalid, an error message will display. If the email address is valid,a message will display and you will receive an email with your username. 
  3. If you click the Reset button, the email field will be reset.
