Manage Sub Terms

Navigation: Select Reference Module->Manage Sub Terms

Using this page, you can view, add, edit and delete Sub Terms in the system.  

Add New Sub Term

  1. Click the icon.
  2. Fill the following field(s):
    • Term, select from the list. Reference data of Term should be predefined.
    • Start, use to select date.
    • End, use to select date.
    • Sub Term
  3. Click the Save button to continue the process or click Cancel to cancel the add operation.
Note: Fields marked with * are mandatory.

Edit Sub Term

  1. Select a Sub Term from the list to edit. The selected row is highlighted in yellow. 
  2. Click the icon of the Sub Term.
  3. Edit the data
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Sub Term

Note: Data shared by other modules cannot be deleted.
  1. Select a Sub Term from the list to delete. The selected row is highlighted in yellow. 
  2. Click the icon of the Sub Term.
  3. A confirmation message is displayed. Click OK to confirm the deletion or click cancel to cancel the deletion.
