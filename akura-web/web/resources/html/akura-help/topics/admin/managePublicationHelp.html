Manage News and Events

Navigation: Select Reference Module->Manage News and Events

Using this page, you can view, add, edit and delete news and events in the system.

View New News And Events

This is the home page for the News & Events section. It displays a list of news and events recorded in the system.

Add New News And Events

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Provide the following details.
    • Header
    • Message
    • Upload Image
  3. Provide an expire date using the icon.
  4. Select the type.
  5. Click the Save button to save the details or click Cancel to cancel the add operation.

    Edit News and Events

    1. Select a News and Events item from the list to edit. The selected row is highlighted in yellow. 
    2. Click the icon of the News and Events item.
    3. Edit the data.
    4. Click the Save button to save the changes or click Cancel to cancel the changes. 

    Delete News and Events

    1. Select a News and Events ion from the list to delete. The selected row is highlighted in yellow. 
    2. Click the icon of the News and Events item.
    3. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
