Change Password

Navigation: Change Password prompt appears in the initial login.
Using this page, you can change the default password set by the administrator. 

Note : Only an administrator can change user passwords.

  1. To change the password, enter the following details:
    • Old Password - Enter the default password emailed to you by the administrator. 
    • New Password - This should be the password that you want to use in the future.
    • Confirm Password - Enter your new password again to confirm.
  2. Click the Next button. 
  3. If the password is successfully changed, you will be logged in to AKURA and an email will be sent to your email account.
