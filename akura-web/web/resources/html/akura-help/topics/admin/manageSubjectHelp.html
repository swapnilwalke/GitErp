Manage Subjects

Navigation: Select Reference Module->Manage Subjects

Using this page, you can search and then add, edit and delete subjects in the system.

Search Subject

  • Enter the full name or partial name of a subject.
  • Click the Search button to get the search results. 

View Subject Details

This is the home page for the Subject section which displays a list of subjects available in the system.

Add New Subject

Note: Fields marked with * are mandatory.
  • Click the button.
  • Enter the subject name.
  • Provide a short name. 
  • Select a stream.
  • Provide the subject code given by the government.
  • Click the Save button to add the subject or click Cancel to cancel the add operation. 

Edit Subject

  • Click the button. The selected record is highlighted in yellow.
  • Edit the subject name, short name, code and the stream.
  • Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Subject

  • Click the button. The selected record is highlighted in yellow.
  • A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
