Manage Staff Categories

Navigation: Select Reference Module->Manage Staff Service Category

Using this page, you can view staff category details and add, edit and delete staff categories in the system.

Staff Category Details

This is the home page for the Staff Category section. It displays a list of staff categories available in the system.

Add New Staff Category 

Note: Fields marked with * are mandatory.
  1. Click the Add New Staff Category button to go to the Add New Staff Category page (bottom of the current page).
  2. Enter the staff category.
  3. Select a staff type by clicking its radio button.
  4. Click the Save button to add the staff category or click the Cancel button to go to the Staff Category Details page.

Edit Staff Category 

  1. Click the Edit Staff Category button to go to the Edit Staff Category page (bottom of the current page). The selected row is highlighted in yellow. 
  2. Edit the staff category and the staff type.
  3. Click the Save button to save the changes or click on the Cancel button to go to the Staff Category Details page.

Delete Staff Category 

Note: Data shared by other modules cannot be deleted.
  1. Select a staff category from the list to delete. The selected row is highlighted in yellow. 
  2. Click on the delete button. 
  3. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to go to the Staff Category Details page.
