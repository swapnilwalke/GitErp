Manage Districts

Navigation: Select Reference Module->Manage Districts

Using this page, you can search for a district and then add, edit and delete districts in the system.

Search District

  1. Enter the full name or partial name of a district.
  2. Click the Search button to generate the search results. 

View District Details

This is the home page for the District section. It displays a list of districts available in the system.

Add New District

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the district name.
  3. Select a province from the drop down menu.
  4. Click Save button to add the district.
  5. Click the Cancel button to go to the district details page.

Edit District

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the name of the district and change the province as required.
  3. Click the Save button to save the changes.
  4. Click Cancel to go to the District Details page.

Delete District

Note: Data shared by other modules cannot be deleted.
  1. Click the button. A confirmation message is displayed. The selected row is highlighted in yellow. 
  2. Click OK to confirm the deletion or click Cancel to go to the District Details page.
