School Module

The School Module allows you to edit the below information of a school.

  • School Name
  • School Address 
  • Email Address
  • Contact No.
  • Fax No.
  • Web Site
  • Start Date
  • Country
  • Province
  • District 
  • Principal
  • Vice Principal
  • Number of Staff
  • Number of Students
  • Primary and Secondary school information
  • Facilities provided by the school
