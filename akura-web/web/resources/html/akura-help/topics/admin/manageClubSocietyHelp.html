Manage Clubs & Societies

Navigation: Select Reference Module->Manage Clubs & Societies

Using this page, you can view, add, edit and delete Clubs & Societies.

View Clubs & Societies

This is the home page for the Clubs & Societies section. It displays a list of clubs and societies available in the system.

Add New Club or Society

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Fill the following fields:
    • Club or Society Name
    • Description
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 

Edit Club or Society

  1. Select a club or society from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the club or society.
  3. Edit the Club or Society data.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Club or Society

  1. Select a club or society from the list to delete. The selected row is highlighted in yellow. 
  2. Click the icon of the club or society.
  3. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
