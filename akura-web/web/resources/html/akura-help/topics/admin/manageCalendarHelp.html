Manage Holidays

Navigation: Select Reference Module->Manage Holidays

Using this page, you can view, add, edit and delete holidays of a school year.

View Holidays

This is the home page for the Holidays section. It displays a list of holidays available in the system. Select the year to view the list of holidays for that particular year.

Add New Holiday

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. The Add New Holiday form is displayed at the bottom of the current page.
  3. Fill the following fields:
    • Start Date
    • End Date
    • Reason
  4. Click the Save button to save the holiday or click Cancel to cancel the addition. 

Edit Holiday

  1. Select the relevant holiday from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the holiday.
  3. The Edit Holiday form is displayed. Enter the start date, end date and reason and click the Save button to save the changes or click Cancel to cancel the changes.

Delete Holiday

Note: Data shared by other modules cannot be deleted.
  1. Select a holiday from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the Holiday.
  3. A confirmation message is displayed. Click OK to delete the holiday or click Cancel to cancel the deletion.
