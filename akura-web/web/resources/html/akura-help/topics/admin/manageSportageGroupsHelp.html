Manage Sports Age Groups

Navigation: Select Reference Module->Manage Sports Age Groups

Using this page, you can view, add, edit and delete sports age groups in the system.

View Sports Age Details

This is the home page for the Sports age section. It displays a list of aports age categories available in the system.

Add New Sports Age Group

Note: Fields marked with * are mandatory.
  1. Click the  icon.
  2. Fill the following field:
    1. Sports Age Group
  3. Click the Save button to save the details or click Cancel to cancel the add operation.

Edit Sports Age Group

  1. Select a sport age group from the list to edit. The selected row is highlighted in yellow. 
  2. Click the  icon of the sports age group.
  3. Edit the sports age group field.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sports Age Group

Note: Data shared by other modules cannot be deleted.
  1. Select a sport age group from the list to delete. The selected row is highlighted in yellow. 
  2. Click the  icon of the sports age group.
  3. A confirmation message is displayed. Click OK to delete or click Cancel to cancel the deletion.
