Manage System Users

Navigation: Select User Management Module->Manage System Users

Using this page, you can search, add, edit, unlock, enable/disable and delete users in the system. 

Search System User

  1. Enter the full name or part of a user name.
  2. Select the user role from the User Role drop down list.
  3. Click the Search button to get the search results. 

Add New System User

Note: Fields marked with * are mandatory.
  • The system redirects to the System User Details page.

Edit System User 

  • The system redirects to the System User Details page and displays the selected details. 

Locked Users

If you enter an incorrect password more than 3 times, your user account will get locked.

  • Locked users appears with the icon.

Unlock User 

Note: Only administrators can unlock users.
  1. Click the  button.
  2. Click OK to unlock the user. 
  • Unlocked users appears with the icon.

Enable/Disable User 

  1. Click  the (enable) or the (disable) button.
  2. Click OK to enable or disable the selected user.

Delete User

  1. Click the button. The selected row is highlighted in yellow. 
  2. Click OK to confirm the deletion or click Cancel to cancel the deletion.
Note: Administrators can only delete disabled/locked users.
