Manage Staff Service Category

Navigation: Select Reference Module->Manage Staff Service Category

Using this page, you can view, add, edit and delete staff service categories in the system.

View Staff service category Details

This is the home page for the Staff service category section. It displays a list of staff service categories in the system.

Add New Staff Service Category

Note: Fields marked with * are mandatory.
  1. Click the  icon.
  2. Fill the following field:
    • Staff Service Category Name
  3. Click the Save button to save the details or click Cancel to cancel the add operation.

Edit Staff Service Category

  1. Select a staff service category from the list to edit. The selected row is highlighted in yellow. 
  2. Click the  icon of the staff service category.
  3. Edit the staff service category data.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Staff Service Category

  1. Select a staff service category from the list to delete. The selected row is highlighted in yellow. 
  2. Click the  icon of the staff service category.
  3. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 
Note: Data shared by other modules cannot be deleted.
