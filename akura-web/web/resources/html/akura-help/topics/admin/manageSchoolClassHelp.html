Manage Classes

Navigation: Select Reference Module->Manage Classes

Using this page, you can view, add, edit and delete classes in the system.

View Class Details

This is the home page for the Class section. It displays a list of classes available in the system.

Add New Class

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Fill the following field:
    • Class
  3. Click the Save button to save the details or click Cancel to cancel the add operation. 

Edit Class

  • Select a Class from the list to edit. The selected class is highlighted in yellow. 
  • Click the icon of the relevant Class.
  • Edit the data.
  • Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Class 

  • Select a Class from the list to delete. The selected class is highlighted in yellow. 
  • Click the icon of the relevant Class.
  • A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion. 

Note: Data shared by other modules cannot be deleted.
