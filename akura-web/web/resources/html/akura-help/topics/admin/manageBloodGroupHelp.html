Manage Blood Groups

Navigation: Select Reference Module->Manage Blood Groups 

Using this page, you can view, add, edit and delete blood groups.

The existing list of Blood Groups will be loaded for the first time. 

Add New Blood Group

  1. Click the icon.
  2. Enter the blood group.
  3. Click the Save button to save the changes or click Cancel to cancel the changes. 
Note: Fields marked with * are mandatory.

Edit Blood Group

  1. Select a Blood Group from the list to edit. The selected row is highlighted in yellow. 
  2. Click the icon of the relevant blood group. 
  3. Edit the data.
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Blood Group

Note: Data shared by other modules cannot be deleted.
  1. Select a Blood Group from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the relevant blood group.
  3. A confirmation box will be displayed. Click OK to continue delete process or click Cancel to cancel the deletion.
