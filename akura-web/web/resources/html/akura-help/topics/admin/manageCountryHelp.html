Manage Countries

Navigation: Select Reference Module->Manage Countries

Using this page, you can search, view, add, edit and delete country related information in the system. The storage of country related information is important to identify the country in which the school is located.

Search a Country

  1. Enter the full name/partial name of a country.
  2. Click the Search button to view the search results. 

View Country Details

This is the home page for the Country section. It displays a list of countries available in the system.

Add New country

Note: Fields marked with * are mandatory.
  1. Click the Add New Country button.
  2. Enter the name of the country. 
  3. Click the Save button to add the country or click the Cancel button to go to the Country Details page.

Edit country

  1. Click the button. The selected row is highlighted in yellow. 
  2. Click the Save button to save the changes.
  3. Click Cancel to go to the Country Details page.

Delete country

Note: Data shared by other modules cannot be deleted.
  1. Click the button. A confirmation message is displayed. 
  2. Click OK to confirm the deletion or click Cancel to go to the Country Details page.
