Manage Races

Navigation: Select Reference Module->Manage Races

Using this page, you can view, add, edit and delete race related information in the system. 

View Race Details

This is the home page for the Race section. It displays a list of races available in the system.

Add New Race

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the race.
  3. Click the Save button to add the Race.
  4. Click the Cancel button to cancel the add operation.

Edit Race

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the race.
  3. Click the Save button to save the changes or click the Cancel button to cancel the changes.

Delete Race

Note: Data shared by other modules cannot be deleted.
  1. Click the button. The selected row is highlighted in yellow. 
  2. Click OK to confirm the deletion or click Cancel to cancel the deletion.
