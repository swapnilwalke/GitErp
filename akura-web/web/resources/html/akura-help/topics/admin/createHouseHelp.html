Manage House

Navigation: Select Reference Module-> Manage House

Using this page, you can view, add, edit and delete reference data of houses.

View House Details

This is the home page for the House section. It displays a list of houses available in the system.

Add New House

  1. Click the icon.
  2. Fill the following fields:
    • House
    • Color
    • Description
  3. Click the Save button to save the details or click Cancel to cancel the add operation.
Note: Fields marked with * are mandatory.

Edit House

  1. Select a house from the list to edit.
  2. Click the icon of the house.
  3. Edit the data
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete House

  1. Select a house from the list to delete.
  2. Click the icon of the house.
  3. A confirmation message is displayed. Click OK to delete the selected house or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
