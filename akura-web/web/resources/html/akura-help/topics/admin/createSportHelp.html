Manage Sports

Navigation: Select Reference Module->Manage Sports

Using this page, you can view, add, edit and delete sports in the system.

View Sports Details

This is the home page for the Sports section. It displays a list of sports available in the system.

Add a new Sport

  1. Click the icon.
  2. Enter the name of the sport.
  3. Click the Save button to save the details or click Cancel to cancel the add operation. 

Note: Fields marked with * are mandatory. 

Edit Sport

  1. Select a sport from the list to edit. The selected row is highlighted in yellow. 
  2. Click the icon of the sport.
  3. Edit the Sport field.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sport

  • Select a sport from the list to delete.
  • Click the icon of the sport.
  • A confirmation message is displayed. Click OK to delete the selected sport or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
