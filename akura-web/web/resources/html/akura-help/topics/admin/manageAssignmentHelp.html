Manage Assignments

Navigation: Select Reference Module->Manage Assignment

Using this page, you can view, add, edit and delete assignments.

View Assignment Details

This is the home page for the Assignment section. It displays a list of assignments available in the system.

Add New Assignment

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Provide the following details:
    • Select the grade.
    • Select the subject.
    • Select the submission date using the calendar icon. 
    • Enter the assignment name.
    • Enter the assignment description.
    • Select the assignment mark type.
  3. Click the Save button to add the new assignment or click Cancel to cancel the process.

Edit Assignment

  1. Select an assignment from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the assignment.
  3. Edit the data set.
  4. Click the Save button to save the changes or click Cancel to cancel the changes. 

Delete Assignment

Note: Data shared by other modules cannot be deleted.
  1. Select the assignment from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the assignment.
  3. A confirmation message is displayed. Click OK to delete the selected assignment or click Cancel to cancel the deletion.
