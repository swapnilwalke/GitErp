View Audit Details

Navigation: Select Reference Module-> View Audit Details 
Using this page, you can view audit details.

View Audit Details

  1. Provide the following details:
    • Select the user.
    • Enter the business function.
    • Select the From Date.
    • Select the event type.
    • Enter the name of the module.
    • Select the To Date.
  2. Click the Search button to view the search results.
  3. Click the View Details icon to view more details about the selected audit record.
