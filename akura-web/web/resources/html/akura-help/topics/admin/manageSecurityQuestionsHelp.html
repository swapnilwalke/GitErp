Manage Security Questions For Users

Using this page, you can manage security questions.

  1. Enter the following details.
  • Security Question No.1 - Select a security question from the drop down.
  • Answer to the security Question No. 1 - Provide an answer for the selected security question.
  • Security Question No.2 - Select a security question from the drop down.
  • Answer to the security Question No. 2 - Provide an answer for the selected security question.
  1. Click the Next button to continue or click the Reset button to reset the fields.
Note:
  • Users can set security questions and answers in the initial login.
  • All fields are mandatory.
