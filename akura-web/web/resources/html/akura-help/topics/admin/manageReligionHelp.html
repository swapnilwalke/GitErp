Manage Religions

Navigation: Select Reference Module->Manage Religions

Using this page, you can view, add, edit and delete religion related information in the system. 

View Religion Details

This is the home page for the Religion section. It displays list of religions available in the system.

Add New Religion

Note: Fields marked with * are mandatory.
  1. Click the button.
  2. Enter the religion name.
  3. Click the Save button to add the religion or click the Cancel button to cancel the add operation.

Edit Religion

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the religion name.
  3. Click the Save button to save the changes or click the Cancel button to cancel the changes. 

Delete Religion

Note: Data shared by other modules cannot be deleted.
  1. Click the button. A confirmation message is displayed. 
  2. Click OK to confirm the deletion or click Cancel to cancel the deletion.
