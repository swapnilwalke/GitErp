Manage Exam Subjects

Navigation: Select Reference Module->Manage Exam Subjects

Using this page, you can view Exam Subjects and then add, edit and delete subjects. 

View Exams Details

This is the home page for the Exam Subjects section. It displays a list of exam subjects available in the system.

Add New Exam Subject

Note: Fields marked with * are mandatory.
  1. Click the button. 
  2. Select the exam from the dropdown list.
  3. Select subjects from the list box.
  4. Click the Save button to add the exam subject.
  5. Click Cancel to go to the Exam Subject Details page.

Edit Exam Subject

  1. Click the button. The selected row is highlighted in yellow. 
  2. Edit the exam and the subjects.
  3. Click the Save button to save the changes. 
  4. Click the Cancel button to go to the exam subject details page.

Delete Exam Subject

Note: Data shared by other modules cannot be deleted.
  1. Click the button. A confirmation message is displayed. The selected row is highlighted in yellow. 
  2. Click OK to confirm the deletion or click Cancel to cancel the deletion.
