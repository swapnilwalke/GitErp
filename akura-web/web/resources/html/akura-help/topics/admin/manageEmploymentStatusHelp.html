Manage Employment Status

Navigation: Select Reference Module->Manage Employment Status 

Using this page, you can view, add, edit and delete employment status related information in the system.

View Employment Status

This is the home page for the Employment Status section. It displays a list of employment statuses available in the system.

Add New Employment Status

Note: Fields marked with * are mandatory.
  1. Click the icon.
  2. Fill the following field:
    • Employment Status
  3. Click the Save button to add the employment status or click Cancel to cancel the add operation. 

Edit Employment Status

  1. Select an employment status from the list. The selected row is highlighted in yellow. 
  2. Click the icon of the particular employment status.
  3. Edit the data.
  4. Click Save to save the changes or click Cancel  to cancel the changes.

Delete Employment Status

Note: Data shared by other modules cannot be deleted.
  1. Select an employment status from the list to delete. The selected row is highlighted in yellow. 
  2. Click the icon of the particular employment status.
  3. A confirmation message is displayed. Click OK to delete the employment status or click Cancel to cancel the deletion.
