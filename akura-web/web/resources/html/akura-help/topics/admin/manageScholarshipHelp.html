Manage Scholarships

Navigation: Select Reference Module->Manage Scholarships

Using this page, you can view, add, edit and delete scholarship related entries in the system.

View Scholarship Details

This is the home page for the Scholarships section. It displays list of scholarships available in the system.

Add New Scholarship

Note: Fields marked with * are mandatory.
  1. Click the  icon.
  2. Fill the following field:
    • Scholarship
  3. Click the Save button to save the details or click Cancel to cancel the add operation.

Edit Scholarship

  1. Select a scholarship from the list to edit. The selected row is highlighted in yellow. 
  2. Click the  icon of the relevant scholarship.
  3. Edit the Scholarship field.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Scholarship

Note: Data shared by other modules cannot be deleted.
  1. Select a scholarship from the list to delete. The selected row is highlighted in yellow. 
  2. Click the  icon of the scholarship.
  3. A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
