Manage Exams

Navigation: Select Reference Module->Manage Exams

Using this page, you can view, add, edit and delete exam related data in the system.

View Exams Details

This is the home page for the Exams section. It displays a list of exams available in the system.

Add New Exam 

  • Click the icon.
  • Select the grade relevant for the exam. 
  • Enter the name of the exam.
  • Select the mark type as grade or marks. 
  • Click the Save button to save the changes or click Cancel to cancel the changes. 

Notes

  • Fields marked with * are mandatory.
  • When adding an exam it should be separated for different streams, eg. for A/L , A/L Science, A/L Commerce, A/L Arts.
  • Then from the Exam Subjects UI allocate subjects for those exams separately.

Edit Exam

  • Select an exam from the list. The selected row is highlighted in yellow. 
  • Click the icon of the relevant exam.
  • Edit the data.
  • Click Save to save the changes or click Cancel to cancel the changes. 

Delete Exam

Note: Data shared by other modules cannot be deleted.
  • Select an exam from the list. The selected row is highlighted in yellow. 
  • Click the icon of the relevant exam.
  • A confirmation message is displayed. Click OK to confirm the deletion or click Cancel to cancel the deletion.
