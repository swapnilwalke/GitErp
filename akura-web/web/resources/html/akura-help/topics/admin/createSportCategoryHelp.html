Manage Sports Categories

Navigation: Select Reference Module->Manage Sports Categories

Using this page, you can view the sports categories and then add, edit and delete sports categories.

View Sport Category Details

This is the home page for the Sports Categories section. It displays a list of sports categories available in the system.

Add New Sport Category

Note: Fields marked in * are mandatory.
  1. Click the icon.
  2. Select the following field:
    • Name of the sport
    • Name of the sport age group
  3. Click the Save button to add the new sport category or click Cancel the process.

Edit Sports Category

  1. Select a sport category from the list.
  2. Click the icon of the selected sport category.
  3. Edit the Sport field and the Sports Age Group field.
  4. Click the Save button to save the changes or click Cancel to cancel the changes.

Delete Sport Category

  1. Select a sports category from the list.
  2. Click the icon of the selected sports category.
  3. A confirmation message is displayed. Click OK to delete the selected sports category or click Cancel to cancel the deletion.
Note: Data shared by other modules cannot be deleted.
