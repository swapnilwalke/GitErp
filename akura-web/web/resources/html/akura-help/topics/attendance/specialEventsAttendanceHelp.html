Special Events Attendance

Navigation: Select Attendance Module->Special Events Attendance

Using this page, you can manually enter attendance for special events. 

  1. Select the following details:
    • Special event
    • Participant
  2. Click the Search button to view a list of participants.
  3. Mark attendance using the check boxes. If necessary, click the Check All button to select all the check boxes.
  4. Click Save to save the changes or click Cancel to cancel the changes.
Note:
  • Fields marked with * are mandatory.
  • Suspended students are listed to be marked for attendance.
  • Past students and students on temporary leave are not listed to be marked for attendance.
