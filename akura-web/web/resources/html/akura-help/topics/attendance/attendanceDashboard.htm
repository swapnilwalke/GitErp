Attendance Dashboard

Navigation: Select Attendance Module-> Attendance Dashboard

Using this page, you can search best student attendance records by grade, class, month and year.

  1. Select the following details:
    • Grade
    • Class
    • Year
    • Month
  2. Click the Search button to view best student attendance.
Notes:
  • Fields marked with * are mandatory.
  • Before generating the report, it is necessary to record holidays and attendance information from the reference module and attendance modules, respectively.
  • To be recognized on the dashboard, eligible students must fullfil the following criteria. 
    • Monthly
      • Academic days for a month should be equal or exceed 10 days.   
      • Student attendance should be greater than or equal to 60% of the number of academic days for a month. 
      • Students must be among the top 10 attendants. 
    • Yearly
      • Student attendance should be greater than or equal to 60% of the number of academic days for a year. 
      • Students must be among the top 10 attendants.
