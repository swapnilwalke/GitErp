Student Attendance

Navigation: Select Attendance Module->Student Attendance

Using this page, you can manually enter class-wise student attendance.

Student Attendance

  1. Select the following details:
    • Class
    • Date
  2. Click the Search Class button to view a list of students for that class.
  3. Mark attendance using the check boxes. If necessary, click the Check All button to select all the check boxes.
  4. Click Save to save the changes or click Cancel to cancel the changes.
Note:
  • Past students, suspended students and students on leave are not listed to be marked for attendance.
  • Fields marked with * are mandatory.
  • Attendance can only be marked for the current date and four dates before the current date. 
  • Dates does not include weekends or holidays. 
  • Holidays should be defined through the reference module, before marking attendance.
