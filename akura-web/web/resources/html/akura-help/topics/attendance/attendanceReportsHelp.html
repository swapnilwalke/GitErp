Daily Attendance

Navigation: Select Attendance Module->Student Attendance

Using this page, you can view class-wise attendance.

  1. Enter the following details:
    • Select the class.
    • Select the date.
  2. Click the Search button to view the list of students for the selected class.
  3. Mark attendance using the check boxes.
  4. Click the Save button.
Note: Fields marked with * are mandatory.
