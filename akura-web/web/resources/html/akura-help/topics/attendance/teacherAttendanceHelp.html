Navigation: Select Attendance Module->Staff Attendance

Using this page, you can manually enter staff attendance.

Staff Attendance

  1. Select the following details:
    • Staff type
    • Date
  2. Click the Search button to view a list of staff members.
  3. Mark attendance using the check boxes. 
    • Click the Check All button to select all the check boxes.
  4. Click Save to save the changes or click Cancel to cancel the changes.
Note:
  • Fields marked with * are mandatory.
  • Terminated staff members are not listed to be marked for attendance
  • Staff members on leave are highlighted in green. 
  • Staff members with a leave process in progress are highlighted in yellow.
  • Attendance can only be marked for the current date and four dates before the current date.
  • Attendance should not be marked for staff members on Full Day leave. System will record their attendance as a Full Day leave. 
  • Attendance should be marked for staff members on Half Day leave. System will record their attendance as a Half Day leave. 
  • Dates does not include weekends or holidays. 
  • Holidays should be defined through the reference module, before marking attendance.
