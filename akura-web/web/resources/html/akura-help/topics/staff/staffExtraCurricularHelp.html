Extra-Curricular Activities

Navigation: Select Staff Module->Search Staff Member->Edit->Extra Curricular

Using this page, you can manage the extra-curricular activity information for staff members.

Notes:

  • If there are any existing extra-curricular activity details, the data will be displayed in the page.
  • Fields marked with * are mandatory.
  • Before adding the Club/Society, Position, Sport or Seminar, make sure that the required reference data has been entered to the system through the Reference module.
  • Clicking the Home button will take you to the Home page.
  • Clicking the Staff Search button will take you to the Staff Search page.

Clubs & Societies

Add Clubs & Societies

  1. To add clubs & societies, click the Add icon and fill in the required details.
  2. Click Save to save the details or click Cancel to cancel the add operation.

Edit Clubs & Societies

  1. To edit clubs & societies, click the Edit icon in the row section.
  2. Make changes as required.
  3. Click Save to save the changes or click Cancel to cancel the changes.

Delete Clubs & Societies

  1. To delete clubs & societies, click the Delete icon in the row section.
  2. A confirmation message is displayed. Click OK to delete the education qualification information or click Cancel to cancel the deletion.
