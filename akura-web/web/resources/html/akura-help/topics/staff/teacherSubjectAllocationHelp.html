Teacher Subject Allocation

Navigation: Select Staff Module->Teacher Subject Allocation

Using this page, you can allocate subjects and classes to a teacher. You can also edit or delete allocations.

  1. Enter the following details:
    • Last name or part of the last name (not mandatory)
    • Employee number (not mandatory)
  2. Click the Search button to view a list of teachers.

Add New Teacher Subject Allocation

  1. Click the icon in the Teacher Subject Allocation page.
  2. Provide the following details:
    • Enter the name of the teacher.
    • Enter the subject and the grade the teacher will teach.
    • Select the Classes check boxes to specify the classes the teacher will teach.
    • Select the year.
  3. Click Save to save the details or click Cancel to cancel the allocation.

Notes:

  • Fields marked with * are mandatory.
  • If the teacher is not listed, you can add the teacher from the Add New Staff Member option in the Staff Member Home page.
  • If the grade and subject are not listed, you can add the grade and subject from the Manage Grade Subject option in the Reference module.
  • If the class is not displayed, you can add the class from the Manage Grade option in the Reference module.
  • Edit and Delete options will not be available for past staff members.

Edit Teacher Subject Allocation

  1. Use the Search feature to find the relevant teacher.
  2. Click the icon of the Particular Teacher to edit teacher subject allocation details.
  3. Change the teacher name, grade, subjects or classes as required.
  4. Click Save to save the changes or click Cancel to cancel the changes.

Delete Teacher Subject Allocation

  1. Use the Search feature to find the relevant teacher.
  2. Click the icon of the relevant allocation.
  3. A confirmation message is displayed. Click OK to delete the selected allocation or click Cancel to cancel the deletion.
