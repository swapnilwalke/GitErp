Section Head Allocation

Navigation: Select Staff Module-> Section Head Allocation

Using this page, you can allocate a Section Head. You can also edit or delete allocations.You can search for a staff member using the staff member type, staff member last name or staff member employee number.

  1. Enter any of the following details:
    • Last name or part of the last name 
    • Employee number 
    • Year (default value is the current year)
  2. Click the Search button to view the results.

Add New Section Head Allocation

  1. Click the icon in the Section Head Allocation page.
  2. Provide the following details:
    • Select the name of the teacher
    • Select the section or select the grade.
    • Select the start date and end date.
  3. Click Save to save the details or click Cancel to cancel the allocation.

Notes:

  • Fields marked with * are mandatory.
  • If the teacher is not listed, you can add the teacher from the Add New Staff Member option in the Staff Module page.
  • Edit and Delete options will be available for for future allocations (When the Start Date/End Date are future dates).
  • Edit and Delete options will not be available for past staff members and for past allocations (When the Start Date/End Date are past dates).
  • Delete option will not be available for current allocations ( When the Start Date is a past and and the End Date is future date).

    Edit Section Head Allocation

    1. Use the Search feature to find the relevant section head.
    2. Click the icon of the relevant section head to edit section head allocation details.
    3. Edit the allocation details as required.
    4. Click Save to save the changes or click Cancel to cancel the changes.

    Delete Section Head Allocation

    1. Use the Search feature to find the relevant section head.
    2. Click the icon for the relevant allocation.
    3. A confirmation message is displayed. Click OK to delete the selected allocation or click Cancel to cancel the deletion.
