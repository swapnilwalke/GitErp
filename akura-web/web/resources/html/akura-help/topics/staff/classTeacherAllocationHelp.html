Class Teacher Allocation

Navigation: Select Staff Module-> Class Teacher Allocation

Using this page, you can search for a class teacher allocation and then add, edit or delete allocations.

  1. Enter the following details:
    • Last name or part of the last name (not mandatory)
    • Class Grade(not mandatory)
    • Year (default value is the current year)
  2. Click the Search button to view the search results.

Add Class Teacher Allocation

  • Click icon in the Class Teacher Allocation page.
  • Select the name of the teacher.
  • Select the Grade Class.
  • Select the year.
  • Click Save to save the details or click Cancel to cancel the allocation.

Notes:

  • Fields marked with * are mandatory.
  • If the teacher is not listed, you can add the teacher from the Add New Staff Member option in the Staff Member Home page.
  • If the Grade Class is not displayed, you can add the Grade Class from the Manage Grade option in the Reference module.
  • Edit and Delete options will not be available for past staff members.

Edit Class Teacher Allocation

  1. Use the Search feature to find the relevant staff member.
  2. Click the icon for the relevant teacher to edit allocation details.
  3. Edit the allocation details as required.
  4. Click Save to save the changes or click Cancel to cancel the changes.

Delete Class Teacher Allocation

  1. Use the Search feature to find the relevant staff member.
  2. Click the icon for the relevant allocation.
  3. A confirmation message is displayed. Click OK to delete the selected allocation or click Cancel to cancel the deletion.
