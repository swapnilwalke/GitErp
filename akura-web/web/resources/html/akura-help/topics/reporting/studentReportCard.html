Student Report Card

Navigation: Select Reporting Module->Student General Reports->Student Report Card

Using this page, you can generate the report card for a selected student.

  1. Provide the following details:
    • Select a class.
    • Select the student admission number.
  2. Click the Generate Report button.
Notes:
  • Fields marked with * are mandatory.
  • Mark entry for all subjects should be confirmed from the student module->Student Mark Entry to view the report data.
  • Only the administrator, student's subject teacher, student's class teacher can generate Student Report Card.
