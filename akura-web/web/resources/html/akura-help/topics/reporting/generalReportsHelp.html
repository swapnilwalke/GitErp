Student General Reports

Navigation: Select Reporting Module->Student General Reports

Using this page, you can view the following reports:

  • Student Summary Report - Shows a summarized report of personal details for a selected student.
  • Student Report Card - Shows the report card for a selected student.
  • Grade Wise Term Marks Graph - Generates a report of average term test marks for a selected subject for a selected grade.
  • Grade Wise Student Term Marks Evaluation - Generates the grade-wise student term marks evaluation report.
  • Class Wise Term Exam Absentees - Shows a report of students who were absent for term exams for a selected class and grade.
  • Class Wise Students Report - Shows the student report for a selected class.
  • Class Wise Student Disciplinary Action Report - Shows the student disciplinary action report for a selected class.
  • Class Wise Student Mark Sheet - Shows the student marks sheet for the selected class and grade for a selected year and term.
  • Exam Results Report - Shows exam details for a selected exam. You have to enter the exam index and name to view the report.
  • Prize List - Shows the class wise and student wise report with students' prizes.
  • Student Scholarship Report - Generates a report that lists the recipients of a selected scholarship.
