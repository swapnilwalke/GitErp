Grade-Wise Best Student Attendance Report

Navigation: Select Reporting Module->Student Attendance Reports->Grade-Wise Best  Student Attendance Report

Using this page, you can generate best student attendance report for a particular grade. 

Grade-Wise Best Student Attendance Report

  1. Select the following details:
    • Grade
    • Class
    • Year
    • Month
  2. Click the Generate Report button.
Notes:
  • Fields marked with * are mandatory.
  • To be listed in the grade-wise best student attendance report students must fullfil the following criteria. 
    • Monthly
      • Academic days for a month should be equal or exceed 10 days.   
      • Student attendance should be greater than or equal to 60% of the number of academic days for a month. 
    • Yearly
      • Student attendance should be greater than or equal to 60% of the number of academic days for a year.
