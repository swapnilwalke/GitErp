Staff Attendance Reports

Navigation: Select Reporting Module->Staff Attendance Reports.

Using this page, you can view the following staff attendance reports:

  • Staff-wise Attendance Report - Shows the daily attendance for a selected staff member for a selected time period.
  • Staff-wise Present and Absent Days Report - Shows the present and absent days for a selected staff member for a selected time period.
  • Staff Late Comers Attendees Report - Shows a list of latecomers for a selected time period.
  • Per Day Staff Category-wise Attendance Report - Shows the category-wise staff attendance for a selected date.
