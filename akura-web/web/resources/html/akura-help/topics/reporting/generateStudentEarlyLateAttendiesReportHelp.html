Generate Student Late Attendees Report

Navigation: Select Reporting Module->Student Attendance Reports->Student Late Comers Attendees Report.

Using this page, you can generate the student late attendance report.

  1. Select the following details:
    • Date range
    • Swipe-in times for which the report should be generated
  2. Click the Generate Report button.
Note: Fields marked with * are mandatory.
