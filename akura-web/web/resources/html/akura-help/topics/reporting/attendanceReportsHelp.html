Student Attendance Reports

Navigation: Select Reporting Module->Student Attendance Reports.

Using this page, you can view the following student attendance reports:

  • Student-wise Attendance Report - Shows the daily attendance for a selected student for a selected time period.
  • Student Attendance on Special Events - Shows the attendance for a selected event.
  • Student Late Comers Attendees Report - Shows a list of latecomers for a selected time period.
  • Per Day Class-wise Attendance Report - Shows the student attendance for a selected class and date.
  • Annual Student Attendance Report - Shows the annual student attendance for a class.
  • Grade-Wise Best Student Attendance Report - Shows the best student attendance for a grade.
