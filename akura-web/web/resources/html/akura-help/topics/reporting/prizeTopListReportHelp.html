Prize List Report

Navigation: Select Reporting Module->Student General Reports->Prize List

Using this page, you can generate a list of students who scored top marks for a selected subject or the highest total marks for a selected year.

  1. Select the following details:
    • Grade.
    • Subject. (Optional. You do not have to select this field if you only want to view a list of prizes for a year)
    • Year.
    • Number of prizes.
  2. Click the Generate Report button.
Note:
  • Mark entry for all subjects should be confirmed from the student module for all terms of all classes of a selected grade to generate the report.
  • Fields marked with * are mandatory.
