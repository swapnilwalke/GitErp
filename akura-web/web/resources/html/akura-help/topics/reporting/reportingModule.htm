Reporting Module

The Reporting module allows you to generate/view the following reports:

  • Student General Reports:
    • Student Summary Report
    • Student Report Card
    • Class Wise Average Term Test Marks for a Given Subjects for a Grade
    • Grade Wise Student Term Marks Evaluation
    • Class Wise Term Exam Absentees
    • Class Wise Students Report
    • Class Wise Student Disciplinary Action Report
    • Class Wise Student Mark Sheet
    • Exam Results Report
    • Prize List
  • Student Attendance Reports
    • Student Wise Attendance Report
    • Students Attendance on Special Events
    • Student Late Comers Attendees Report
    • Per Day Class Wise Attendance Report
    • Grade-Wise Best Student Attendance Report
  • Staff General Reports 
    • Staff Profile Report
    • Customized Staff Related Reports
  • Staff Attendance Reports
    • Staff Wise Attendance Reports
    • Staff Wise Present and Absent Days Report
    • Staff Late Comers Attendees Report
    • Per Day Staff Category Wise Attendance Report
