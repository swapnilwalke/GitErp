Customized Staff Reports

Navigation: Select Reporting Module->Staff General Reports->Customized Staff Related Reports.

Using this page, you can view the customized staff related reports.

  1. Select one of the following reports:
    • Staff By Type Report - Shows a list of staff members by type.
    • Section Head List - Shows a list of section heads.
    • Class Teacher List - Shows a list of class teachers by the selected grade.
  2. Click the Generate Report button.
